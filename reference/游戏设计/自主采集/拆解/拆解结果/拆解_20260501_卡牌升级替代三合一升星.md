# 拆解结果：卡牌升级替代三合一升星（王者万象棋等级制）

## 基本信息
- **来源**：../研究/研究结果/研究_20260501_卡牌升级替代三合一升星.md
- **拆解时间**：2026-05-01 08:46 UTC
- **原始游戏**：天美《王者万象棋》
- **适配度**：0.87

---

## 四层拆解

### 第一层：行为层

#### 行为变化对比

| 维度 | 传统三合一升星（原有行为） | 等级制升级（改变后行为） |
|------|---------------------------|--------------------------|
| **同名卡处理** | 囤积同名卡，等凑齐3/9张 | 每张同名卡立即打出，等级+1 |
| **成长节奏** | 阶跃式等待，凑齐→跳跃 | 连续即时成长，每张都有正反馈 |
| **天花板感知** | 3星=终点，达到后无成长 | 无感知天花板，始终有"下一个等级" |
| **资源取舍** | 占背包格子多线追三星 | 等级可转移，灵活调整投入方向 |
| **失败处理** | 追星失败卡牌沦为废卡 | 溢出卡仍有等级价值 |
| **Combo触发** | 被动靠商店刷新 | 主动出牌触发登场/牺牲/整备效果 |

#### 行为本质变化

从「**囤积等待+运气爆破型成长**」变为「**即时投入+持续线性成长+里程碑突破型规划**」

玩家不再是"赌狗"心态（赌第9张），而是"养成"心态（每一张都是经验值）。

---

### 第二层：系统层

#### 被改变的系统

**1. 成长曲线模型重构**

| | 原本 | 改变后 |
|---|---|---|
| 曲线形态 | 阶梯状：2张卡=0提升，第3张=巨大跳跃 | 线状：每张卡=积分式提升 |
| 成长效率 | 存在"死区"（凑卡等待期战力停滞） | 无死区，持续微幅增长 |
| 顶峰结构 | 3星硬天花板 | 999级软天花板+关键等级质变点 |

**2. 资源投入-产出模型改写**

| | 原本 | 改变后 |
|---|---|---|
| 投入确定性 | 不确定（第3张什么时候来？） | 确定（每张卡就是1级） |
| 投入可逆性 | 不可逆（升星后不能拆） | 可逆（等级可转移/互换） |
| 过剩资源处理 | 3星后溢出卡无意义 | 溢出卡持续有升级价值 |

**3. 阵容构筑的约束体系变化**

| | 原本 | 改变后 |
|---|---|---|
| 构筑核心 | 羁绊表（种族/职业人口数） | 关键词Combo链（登场/牺牲/整备联动） |
| 卡片价值 | 主要看羁绊标签 | 看关键词效果+联动可能 |
| 阵容灵活性 | 换阵容代价大（羁绊断裂） | 高灵活（等级可转移，关键词可重组） |

**4. 战斗触发时机系统扩展**

原本触发时机有限（战斗开始/死亡/攻击时等），等级制引入新触发维度：
- 「登场」：打出时即触发
- 「合成」：卡牌合成时触发
- 「整备」：回合开始时触发
- 「牺牲」：阵亡时触发（已有）
- 「退场」：退场时触发

---

### 第三层：决策层

#### 短期决策（战斗中）

1. **出牌节奏决策**：是先打低费卡过渡，还是直奔核心C位升级？（传统：被商店刷新周期限制）
2. **效果牌时机选择**：「换帅」类效果牌何时使用？在对手阵容暴露后？还是紧急挽救被针对的C位？
3. **关键词触发链规划**：这一轮用「登场」流快速爆发，还是「整备」流稳扎稳打？
4. **竞拍博弈**：对关键卡/效果牌，是抬价消耗对手资源，还是让过保留金币？

#### 长期决策（构筑方向）

1. **从"能追谁3星"→"养哪个大哥"**：不再被RNG发牌决定方向，而是主动选择核心C位持续投入等级
2. **宽阵vs深阵**：是把等级资源分散到多个棋子（均衡升级），还是集中到一个C位（极限单核）？等级可转移降低了试错成本
3. **后续卡牌选择逻辑改变**：不只考虑"这张卡是否适配我的羁绊"，更要考虑"这张卡的关键词能否联动我的升级链"
4. **构筑可逆性解放决策**：等级可转移→失败构筑可回收→鼓励玩家冒险尝试非主流Combo

### 第四层：情感层

#### 爽点分析

**1. 每张卡都是"中奖"（持续正反馈）**
- 为什么爽：传统自走棋中，非凑齐张数的同名卡是"废卡"，等级制让每一张卡都有价值
- 场景：D牌时每出一张同名卡，场上战力立刻+1级，不再是"又没凑齐"的失望

**2. 关键等级突破的仪式感**
- 为什么爽：10/40/100级解锁技能形态变化，是精心设计的高光时刻
- 场景：吕布100级解锁全屏击飞→动画华丽且逆转战局，类似RPG的"大招解锁"

**3. 策略自由度的掌控感**
- 为什么爽：等级可转移意味着"我可以重新来过"，不再被一次错误的升级决策绑定
- 场景：中期发现C位选错，用换帅效果卡把等级转移给正确棋子，体验"大翻盘"的智力快感

**4. 线性成长的安心感**
- 为什么爽：不再有"第9张就是不来"的RNG焦虑，每D一次都有进度条式的成长
- 场景：大后期资源溢出时，继续投入已有3星棋子升级到Lv.25，每一张卡都在变强

**5. Combo链爆发的连锁爽感**
- 为什么爽：登场→触发→再触发，关键词层层联动形成"多米诺骨牌"效应
- 场景：打出蔡文姬→同阵营随机三位+1级→触发李信牺牲机制→临时等级+7→瞬间战力碾压

#### 情绪类型

**持续养成+里程碑爆发+策略掌控**（对比传统三合一的「赌徒焦虑+爆破释放」）

## 玩法块公式（抽象）

> **原本**：升星需要凑齐3/9张同名卡才触发阶跃式质变，凑卡等待期战力为零增长，3星后无继续成长空间

> **现在**：每一张同名卡打出即转化为等级经验值，成长曲线从阶梯变为斜率线，关键等级节点解锁技能形态变化，等级作为"可转移资产"在棋子间灵活流动

## MCGG迁移方案

### 羁绊效果

#### 方案A：溢出转化羁绊（轻量级）
```
【经验溢出】(3/5/7)
战斗结束后，场上每个3星棋子获得的溢出同名卡按比例转化为经验值
(3) 每张溢出卡为随机同羁绊棋子+1级
(5) 每张溢出卡为所有同羁绊棋子+0.5级
(7) 溢出卡金额额外转化为全队经验池，可自由分配
```

#### 方案B：关键等级突破羁绊
```
【突破】(2/4)
棋子达到关键等级时解锁额外效果
(2) Lv.10解锁：技能伤害+15%
(4) Lv.25解锁：新增一项被动技能
```

---

### 海克斯/Augment

#### A. 等级转移类
- **换帅**（金色）：每回合可免费将两个同费用棋子的等级互换一次
- **经验分享**（银色）：每当你升级一个棋子时，同羁绊中等级最低的棋子获得该等级提升量的25%
- **等级吸收**（棱彩）：出售棋子时，其等级的50%转化为全局经验池，可分配给其他棋子

#### B. 里程碑突破类
- **突破极限**（棱彩）：3星棋子等级上限提升至50级，且Lv.25/50解锁技能进化效果
- **快速成长**（金色）：每回合开始时，随机一个非3星棋子+3级
- **连胜经验**（银色）：连胜时，场上等级最低的棋子额外+2级

#### C. Combo联动类
- **登场即战力**（金色）：每当你打出棋子时（首次登场），该棋子+3级，并在当前回合获得20%伤害减免
- **牺牲传承**（银色）：棋子阵亡时，其等级的30%转移给存活同阵营棋子
- **整备时刻**（金色）：回合开始时，等级最高的棋子为同阵营其他棋子各+1级

---

### 棋子技能

#### 1. 蔡文姬（辅助-支援型）
**登场效果**：让等级最低的三位同阵营英雄等级+1
**技能**：为周围友军提供护盾，护盾值随自身等级成长（每级+2%）

#### 2. 吕布（战士-Carry型）
**技能**：对前方扇形区域造成伤害
- Lv.10 进化：伤害范围扩大30%
- Lv.40 进化：技能命中额外减少目标30%护甲
- Lv.100 进化：全屏击飞+高额真实伤害

#### 3. 李信（战士-牺牲联动型）
**技能**：阵亡时触发，周围友军获得攻击力加成
- 被动：每当有同伴阵亡，临时等级+7（持续至回合结束）
- 等级越高，阵亡触发效果越强

#### 4. 赵云（战士-登场型）
**登场效果**：自身+随机两位友军提升等级
**技能**：突刺至敌方后排，伤害随等级线性增长

## 核心启示

1. **"每张卡都有用"是降低挫败感的关键**：MCGG可以通过溢出卡转化机制，至少让3星后的同名卡不完全是废卡
2. **线性成长+里程碑爆发**：比单纯的阶跃式升星更能提供持续心流
3. **可逆的资源投入**：等级可转移让玩家更敢于尝试，提升了策略深度而非复杂度
4. **关键词/Effect联动才是灵魂**：等级制本身只是骨架，登场/牺牲/整备等触发联动才是策略趣味的来源
