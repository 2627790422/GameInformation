# 拆解结果：强化果实机制

## 基本信息

- **机制名称**：强化果实（Empowerment Fruits）
- **来源**：../研究/研究结果/研究_20260427_强化果实机制.md
- **拆解时间**：2026-04-27 19:53 UTC
- **拆解者**：锅巴

---

## 四层拆解

### 行为层

**原本行为（玩家）**
- 不再：无条件接受羁绊给定的棋子定位，出装/强化方向完全固定
- 改为：主动选择棋子的个性化强化方向，棋子定位不再唯一
- 可以：用打工棋子试错方向，后期拆卸重新选择
- **行为本质变化**：从"选择棋子"升级为"定制棋子"——玩家决策从宏观（羁绊选择）下沉到微观（单棋子的成长路径）

**原本行为（系统）**
- 不再：羁绊效果直接决定棋子强度，棋子被动接受
- 改为：赋予每个棋子独立的强化回路，与羁绊解耦
- **系统行为本质变化**：从"羁绊驱动"转变为"棋子驱动"——强化系统绕过羁绊直接作用于单棋子

---

### 系统层

**系统1：获取与消耗系统**
- 原本：羁绊等级决定强化效果强度（被动触发）
- 改成：玩家主动消耗资源（回合时机+金币）获取个性化强化，且可拆卸重选

**系统2：效果池系统**
- 原本：羁绊效果池有限且同质（同羁绊棋子强化路径高度雷同）
- 改成：106+种效果 + 每棋子专属池，差异化最大化

**系统3：后悔机制（拆卸刷新）**
- 原本：选择不可逆，一旦确定无法调整
- 改成：拆卸器 + 出售棋子 = 重新获取果实，无限刷新

**系统4：果实羁绊系统**
- 原本：羁绊是唯一的组合效果来源
- 改成：同系列果实两两组合触发协同效果，增加第二层组合维度

---

### 决策层

**短期决策（单局内）**
- 2次使用机会：早期（2-1）建立优势 vs 中期（3-6）调整方向
- 三选一：权衡3个选项的即时收益
- 是否使用拆卸器：重新选择的机会成本
- 是否出售棋子重获取果实：放弃棋子 vs 刷新效果

**长期决策（战略规划）**
- 是否围绕特定果実に构建阵容（选择棋子时考虑果实适配性）
- 效果池分类筛选：避免选择瘫痪（信息管理能力）
- 果实羁绊的组合探索：哪个效果组合能产生1+1>2

---

### 情感层

**爽点1：爆气动画反馈**
- 为什么爽：即时视觉冲击 + "我的棋子变强了"的具象化表现
- 情绪类型：成就感、力量感

**爽点2：专属定制感**
- 为什么爽：每个棋子的强化都是独一无二的，彰显玩家个性
- 情绪类型：占有感、独特性满足

**爽点3：后悔药机制**
- 为什么爽：降低决策压力，允许试错，"这次选错了没关系"
- 情绪类型：安全感、释然

**爽点4：信息差博弈**
- 为什么爽：观察对手的果实选择，猜测其战略方向
- 情绪类型：竞争感、智识优越感

**情绪类型总览**：成就感 + 安全感（降低惩罚） + 个性化满足 + 策略博弈深度

## MCGG迁移方案

### 羁绊效果

**【新增】果实羁绊**
- 触发条件：同系列2个强化果实装备在同一阵容
- 效果示例：
  - 火系果实×2 → 攻击+15%，攻击命中灼烧
  - 水系果实×2 → 法力回复+20%，技能CD-10%
  - 风系果实×2 → 闪避+15%，移动速度+10%
  - 雷系果实×2 → 暴击+20%，暴击伤害+25%

**【羁绊改造】部分羁绊强化效果开放"果实联动"**
- 现有羁绊可额外激活"果实羁绊"条件，享受双重协同

---

### 海克斯/Augment

**【新海克斯】强化果实相关**

| ID | 名称 | 效果 |
|----|------|------|
| AUG_FRUIT_01 | 果实猎人 | 下一个强化果实效果+50% |
| AUG_FRUIT_02 | 双重爆发 | 果实的第3个选项必定是稀有效果 |
| AUG_FRUIT_03 | 拆卸大师 | 拆卸果实不消耗次数，免费重新选择 |
| AUG_FRUIT_04 | 果实亲和 | 所有棋子初始自带1个随机通用果实 |
| AUG_FRUIT_05 | 元素主宰 | 火/水/风/雷果实效果额外+25% |

---

### 棋子技能

**【新通用技能】所有棋子解锁"果实强化"槽位**

**基础机制**：
- 每名棋子可装备1个强化果实
- 效果池分4类：攻击强化 / 防御强化 / 技能强化 / 站位强化

**示例效果池（按类型）**：

| 类型 | 效果名称 | 效果描述 |
|------|----------|----------|
| 攻击 | 锋锐之果 | 攻击力+20%，攻击速度+15% |
| 攻击 | 暴击之果 | 暴击率+25%，暴击伤害+30% |
| 防御 | 铁壁之果 | 护甲+30，魔抗+20 |
| 防御 | 复苏之果 | 生命偷取+15%，脱战回复+10% |
| 技能 | 奥术之果 | 技能伤害+25%，法力上限-15 |
| 技能 | 冷却之果 | 技能CD-20% |
| 站位 | 前排之果 | 受到伤害-15%，嘲讽范围+1 |
| 站位 | 后排之果 | 周围无友军时伤害+25% |

**【专属果实】为MCGG核心棋子设计专属效果**

> 示例（假设棋子：阿狸）
> - 灵魂契约果实：技能额外标记敌人，击杀标记目标刷新技能CD
> - 魅惑绽放果实：大招魅惑时间+0.5秒，范围+1格

**获取时机**：3-1和5-1各1次，拖拽使用，三选一
**拆卸机制**：每阶段1次免费拆卸器，超出消耗5金币

## 总结

**核心迁移思路**：从"羁绊→棋子"的单向设计，转变为"棋子→个性化"的细粒度设计思维。强化果实的本质是**绕过羁绊直接给单棋子注入差异化**，这完美解决MCGG棋子同质化问题。

**迁移优先级**：⭐⭐⭐⭐⭐（最高优先级）
