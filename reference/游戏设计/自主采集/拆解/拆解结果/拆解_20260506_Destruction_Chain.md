# 拆解结果：Destruction Chain 战斗-探索联动循环

## 基本信息
- **来源**：../研究/研究结果/研究_20260505_DK_Bananza_Destruction_Chain.md
- **拆解时间**：2026-05-06 04:34 UTC
- **原游戏**：Donkey Kong Bananza (Nintendo, 2025)

---

## 四层拆解

### 第一层：行为层

#### 行为变化

| 原行为 | 变化后行为 |
|--------|----------|
| 打完怪，手动走到新区域 | 打怪本身破墙，自然暴露新区域 |
| 按箭头/标记导航探索 | 被"破坏欲"驱动，自主选择破坏方向 |
| 拾取固定Power-Up道具获得强化 | 从环境中拾取地形碎片作为临时武器 |
| 强化状态持续到被消耗或关卡结束 | 地形碎片一次性消耗，用完就必须再找 |
| 宝箱/掉落主动给出奖励 | 打破墙壁后被动发现隐藏道具 |
| 战斗和探索是两个独立阶段 | 战斗和探索浑然一体、互相驱动 |

#### 行为本质变化

从"阶段分离式战斗+探索"变为"战斗即探索的一体化循环"——每一次战斗动作都同时产生击杀敌人和改变环境两个结果。玩家被自然的破坏冲动引导，而非被动跟随指引。

---

### 第二层：系统层

#### 1. 战斗-探索耦合模型重构

- **原本**：战斗与探索是串行两阶段——先清怪，再找路
- **转变**：战斗结果直接创造新的探索空间——敌人被击飞撞碎墙壁→暴露隐藏区域→进入新区域→获取新武器
- **系统语言**：将"战斗产出"从单维度（击败敌人）扩展为双维度（击败敌人 + 改变环境）

#### 2. 资源获取模型重构

- **原本**：强化资源（Power-Up）通过固定位置拾取或固定掉落获取，持有后持续生效
- **转变**：强化资源来自环境本身，使用时即时消耗，迫使玩家持续从环境中获取资源
- **系统语言**：从"持有型资源"变为"流转型资源"，创造持续的临场压力

#### 3. 奖励触发方式重构

- **原本**：系统主动给予奖励——宝箱、掉落、任务完成提示
- **转变**：奖励通过玩家行为的自然结果被动暴露——破墙后发现隐藏区域/道具
- **系统语言**：从"系统告知你得到了奖励"变为"你自己发现了奖励"

---

### 第三层：决策层

#### 短期决策（战斗中的即时选择）

- 攻击偏左的敌人还是偏右的敌人？（哪边墙后有隐藏区域？）
- 是否现在使用地形碎片？还是留着等关键敌人？
- 蓄力攻击还是快速攻击？（蓄力攻击破坏范围更大但时间风险更高）
- 先清敌人还是先破坏环境找资源？

#### 长期决策（构筑方向）

- 更看重"破坏环境能力"而非单纯"击杀效率"——改变整个装备/技能优先级
- 攻击速度的价值重新定义——高攻速可以更快破坏墙壁获取资源
- AOE效果获得额外价值——范围攻击同时破坏更多墙壁暴露更多奖励
- "环境互动"类技能从边缘选项变为主力构筑方向
- 防御型道具价值下降——积极破坏环境获取即时资源比苟活更有效

### 第四层：情感层

#### 爽点分析

1. **连锁反应的爽感**：一击产生多重结果——击杀敌人+破坏墙壁+暴露宝藏+获取新武器，一气呵成的反馈链让每次攻击都充满期待。"不是打一下，是引发一场小型雪崩。"

2. **发现的惊喜感**：系统不告诉你奖励在哪，你打破墙壁的那一刻才知道里面有什么。这种"自己找到的"感觉远比"系统给你的"更持久、更有成就感。

3. **破坏的原始快感**：砸东西本身就是一个强烈的正反馈。任天堂利用了人类最本能的破坏欲作为导航系统，玩家不需要地图——砸就对了。

4. **资源即选择的紧张感**：地形碎片用了就没，你必须不断判断"现在用还是留着"。这种即时消耗的压力让每个决策都有分量，但又不会焦虑——因为周围到处都是新资源。

5. **自主创造的掌控感**：不是游戏告诉你"这里有个秘密"，而是你打碎墙壁后大喊"卧槽这里还有个洞！"——玩家将自己的成功归因于自己的探索能力。

#### 情绪类型

破坏本能 + 连锁满足 + 自主发现 = **"破坏→创造→再破坏"的正反馈螺旋**

## 玩法块公式

**原本**战斗和探索是两个独立阶段（打完怪→再找路），**现在**战斗结果直接创造新的探索空间（打怪破墙→暴露区域→获取资源→继续战斗）。

**子公式拆解**：
1. 原本强化状态是固定道具持有，现在强化是环境碎片即时流转
2. 原本奖励由系统主动给予，现在奖励由玩家行为被动暴露
3. 原本导航靠箭头标记，现在导航靠破坏冲动自然引导

## MCGG迁移方案

### 羁绊效果

#### 羁绊：「破坏者」（Destroyer）

**效果**：破坏者棋子击杀敌人时，有概率"击飞"敌人撞向棋盘边缘，对相邻格子造成伤害并留下"地形印记"

```
(2) 破坏者：击杀时有30%概率击飞，对1个相邻格造成50%攻击力伤害
(4) 破坏者：击杀时必定击飞，对2个相邻格造成100%攻击力伤害，留下地形印记
(6) 破坏者：击杀时必定击飞，对周围所有相邻格造成150%攻击力伤害，留下地形印记；战斗结束后地形印记转化为永久地形buff
```

**地形印记效果**：站在被标记的格子上，下一轮获得+10%/20%/30%攻击力（按羁绊等级），持续1回合后消退。

#### 羁绊：「拾荒者」（Scavenger）

**效果**：拾荒者棋子从被击杀的敌人位置"拾取"环境碎片，转化为临时属性

```
(2) 拾荒者：战斗中每击杀一个敌人，获得1层"碎片"（最多10层），每层+2%攻击力，战斗结束清空
(4) 拾荒者：碎片上限提升至20层，每层+3%攻击力+1%攻速；战斗结束保留50%层数至下一轮
```

### 海克斯/Augment

#### 白银级

- **「破墙」**：己方前排棋子首次受到≥自身最大生命值30%的伤害时，击退攻击者1格并破坏该格（使其变为"废墟"地形）。站在废墟上的棋子攻击力-10%。
- **「环境馈赠」**：每轮备战阶段开始时，棋盘随机1格出现"宝藏地形"——部署在该格的棋子在战斗中+15%最大生命值。

#### 黄金级

- **「破坏链」**：己方棋子在3秒内连续击杀2个以上敌方棋子时，随机破坏棋盘上一格（使其不可部署），并对破坏格相邻敌人造成被击杀棋子最大生命值20%的伤害。破坏格在下一轮恢复。
- **「战场伤痕」**：战斗结束后，本场战斗中双方阵亡棋子最多的格子被"标记"。下一轮部署在该格的棋子获得+20%攻击力/+20%生命值，但战斗开始时损失10%当前生命值。
- **「连锁反应」**：当一个羁绊效果触发时，有40%概率额外触发一次随机己方棋子的羁绊效果（不同同名羁绊）。

#### 棱彩级

- **「地形专家」**：备战阶段可以选择棋盘上1个格子"改造"——改造后的格子获得永久地形效果（如：+30%攻击力、+30%生命值、首次攻击晕眩目标1秒）。每轮可以重新选择改造目标，但原改造格效果减半。
- **「永恒伤痕」**：整局对战中，每个被击杀的棋子在其死亡位置留下永久的地形印记。部署在印记上的棋子获得+5%全属性（可叠加，最多5层）。

### 棋子技能

#### 棋子思路：「碎岩巨人」（Rock Breaker）

**定位**：前排坦克/控制

**技能 - 粉碎冲锋**：冲向生命值最低的敌人，对沿途所有敌人造成伤害并"击飞"至棋盘边缘。被击飞撞到棋盘边缘的敌人额外受到一次伤害并被短暂晕眩。如果被击飞的路径经过其他棋子，也对经过的棋子造成伤害。

**设计来源**：Bananza 中"敌人被击飞撞碎墙壁"→ 自走棋中用棋盘边缘代替墙壁。

#### 棋子思路：「寻宝猎人」（Treasure Hunter）

**定位**：后排输出/资源型

**技能 - 环境探测**：战斗中每X秒扫描棋盘，标记一个随机空置格子为"宝藏格"。己方棋子经过或站上宝藏格时获得X%攻击速度加成，持续X秒。

**设计来源**：Bananza 中"破坏墙壁暴露隐藏区域"→ 自走棋中通过技能"暴露"棋盘增益。

#### 棋子思路：「拾荒者」（Scavenger）

**定位**：辅助/成长型

**被动技能 - 环境回收**：每当任意棋子（包括敌方）在战斗中死亡，在死亡位置留下一块"碎片"。拾荒者靠近碎片时自动拾取，每块碎片永久+2攻击力/+10最大生命值（单局内有效，战斗结束清空）。

**设计来源**：Bananza 中"从环境中持续获取即时消耗资源"→ 自走棋中用棋子死亡位置模拟环境资源。

## 关键设计原则提炼

从本次拆解中提炼的可迁移设计原则：

1. **"一次行动，两种产出"**：战斗行为同时产生"击败敌人"+"改变环境"双结果
2. **"资源即选择"**：即时消耗型资源创造紧迫的临场决策空间
3. **"发现 > 给予"**：被动发现比主动奖励更有玩家自主感和成就感
4. **"破坏即导航"**：用玩家的破坏冲动代替箭头引导，探索方向由玩家行为自然决定
5. **"瑕疵服务于体验"**：Bananza 容忍穿模bug因为"对玩家有利"——设计决策的判断标准是玩家体验而非技术完美
