## 四层拆解

### 行为层

**玩家行为变化**：

| 维度 | 没有该系统时 | 有该系统后 |
|------|------------|-----------|
| 高概率决策（>76%） | 畏首畏尾，85%也怕翻车 | 果断押注，知道系统会保 |
| 低概率决策（1-14%） | 直接放弃，5%等于不存在 | 敢赌小概率，15%值得一试 |
| 连续失败时 | 怒退游戏，认为"系统针对我" | 信任系统会兜底，继续打 |
| 连续成功后 | 侥幸心理，随时准备暴毙 | 知道3次后保护消失，开始谨慎 |
| 关底表现 | 全程依赖辅助，突然变难时愤怒 | 前期已存入信任，关底难是"应该的" |
| 失败后的重置 | 心态崩溃，止损退出 | 知道保护已重置，重新燃起希望 |

**行为本质变化**：从"不信概率→愤怒→退出"的负面循环，变为"信任系统→接受偶然→挑战极限"的正向心理旅程。系统在后台管理玩家的概率预期，让行为更接近设计师意图。

---

### 系统层

**被改变的底层规则**：

**① 概率输出模型：从数学真实→感知真实**

```
原本：每次独立Roll点，显示85%意味着15%概率失败
现在：>76%直接判定成功，概率输出与显示值脱钩
```

系统不再输出"真实的数学概率"，而是输出"符合玩家预期的心理概率"。这颠覆了传统RNG的"透明公平"假设。

**② 信任累积系统：从无状态→有状态窗口**

```
原本：每次RNG事件相互独立，系统不跟踪玩家体验
现在：3次保护窗口→释放→失败重置，系统记住了"玩家刚刚经历了什么"
```

RNG从无状态变成了有记忆的有状态系统。这是最大胆的设计——它不再假装"每次随机都是新的开始"，而是承认"玩家的体验是连续的"。

**③ 难度曲线：从线性实际难度→心理难度分层**

```
原本：全流程使用同一套概率规则，难度曲线靠数值增长
现在：前期保护（作弊）→中期真实→终章摘除（纯随机），难度在心理层面分层
```

难度不再是数学曲线的函数，而是玩家心理状态的函数。同样的概率在不同阶段产生不同的心理效果。

**④ 概率地板：切断极端负面体验**

```
原本：5%概率在小样本中几乎可以被视为0%
现在：<15%统一提升到15%，最低限度的"可能发生"被保留
```

极低概率在小样本中创造的不是惊喜而是绝望。15%地板让"理论上可能"变成"偶尔真的会发生"。

---

### 决策层

**短期决策变化**：

- 看到 >76% 成功率时，不再犹豫——决策更果断，节奏更流畅
- 小概率选项（原来15%以下）现在变成可考虑选项，战术池扩大
- 连续成功 2 次后，开始警觉"下一次是真实概率"，引入博弈思考
- 失败后知道系统已重置保护，有动力继续而非退出

**长期决策变化**：

- 前期（1-3次保护）：玩家敢于尝试高风险高回报流派，因为系统在兜底——鼓励探索型构筑
- 中期（释放真实概率）：玩家必须建立不依赖运气的稳定打法，开始认真管理资源——自然完成从"新手保护"到"老手博弈"的过渡
- 终章（训练轮摘除）：玩家必须依靠前期积累的策略深度而非概率保护，此时的失败让玩家反思自己的决策而非质疑系统公平——驱动自我提升而非愤怒退出

---

### 情感层

**爽点分析**：

**① 被保护的安心感**
> "85%终于不会翻车了。"——高概率失败是概率游戏中最致命的负面体验。系统让玩家放心押注，情绪从"不信任"变为"信赖"。

**② 失败后的安全感（反直觉爽点）**
> "虽然刚才失败了，但接下来3次系统会帮我。"——传统RNG失败后玩家感到孤立和受害。Dispatch的失败重置保护让失败从"完蛋了"变成"没事，还有机会"。这是最精妙的情感设计。

**③ 小概率生效的惊喜**
> "5%居然触发了我靠！"——但实际上系统偷偷把它提到了15%。玩家不会觉得被欺骗，只会感到"运气来了"的兴奋。这种善意的欺骗创造了纯粹的快乐。

**④ 终章的真实紧张感**
> "来了，真正的关卡来了。"——前期存入的信任在终章提取。玩家知道这次没有保护了，失败就是失败。紧张感从"不公平的愤怒"转变为"公平对决的刺激"。

**⑤ 节奏呼吸感**
> 保护→释放→失败→保护→释放... 整个体验在"被照顾"和"靠自己"之间呼吸。不是永远的温室，也不是永远的残酷。

**情绪类型**：信任建立 + 被保护的安全感 + 公平对决的紧张 + 善意欺骗的惊喜

## 玩法块公式

```
原本：每次随机事件独立Roll点，概率透明且真实，玩家只能用数学期望做决策

现在：
  1）>76%显示成功率 → 系统自动成功，玩家信任"大概率"
  2）连续3次自动成功后 → 释放真实概率，玩家开始靠自己
  3）一旦失败 → 立即重置3次保护窗口，打断挫败链
  4）<15%显示成功率 → 统一提升到15%，保留"可能发生"的希望
  5）最终章节 → 移除所有隐形辅助，提取前期存入的信任
```

**公式抽象**：RNG从"纯数学函数"变成"情绪管理函数"——输出值 = f(显示概率, 玩家历史体验, 当前阶段)

## MCGG迁移方案

### 羁绊效果

**羁绊：「命运编织者」（Fortune Weaver）**

```
(2) 命运编织者：商店刷新时，与你场上已有羁绊匹配的棋子出现率至少为15%
(4) 命运编织者：商店连续3次未出现"与你最高等级羁绊匹配的棋子"时，第4次必定出现
(6) 命运编织者：每当你的棋子技能触发概率低于15%时，该概率在本次战斗中提升到15%；每次失败后重置该效果
```

**设计思路**：将Dispatch的概率保护机制浓缩为一个羁绊。2/4/6人口逐级解锁完整系统，让玩家在凑羁绊的过程中自然体验"保护→增强"的递进。6人口的"技能触发概率地板"是点睛之笔——让低概率技能棋子在这个羁绊下重获新生。

**羁绊：「命运之轮」（Wheel of Fortune）**

```
(3) 命运之轮：前3回合，所有概率触发效果（技能/装备/海克斯）的实际触发率至少提升10个百分点
     第4回合起，该辅助移除
(5) 命运之轮：失败后，下回合所有概率触发效果获得+15%实际触发率加成（一次性）
```

**设计思路**：直接映射Dispatch的"前期保护→中期释放→失败重置"三段式。3人口给前期窗口保护（对应保护窗口），5人口给失败后的安全网（对应失败重置保护）。这是一个节奏型羁绊，越早凑齐越能安全渡过前期。

### 海克斯/Augment

**海克斯：「概率均衡器」（Probability Equalizer）**

```
类型：全局被动
效果：本局对战中，所有显示概率在76%以上的事件自动视为成功；
     所有显示概率在15%以下的事件实际概率提升到15%。
限制：仅PVE阶段生效（PVP阶段不生效以保持公平竞技）
```

**设计思路**：直接将Dispatch的76%阈值和15%地板搬过来。加PVE限制确保不对PVP公平性造成影响。适合放在银/金色海克斯层，因为效果强大但有趣味性。

**海克斯：「信任训练」（Trust Training）**

```
类型：阶段触发
效果：在阶段1-3期间，商店刷新必定包含至少1个与你场上最高等级羁绊匹配的棋子。
     阶段4开始，该效果移除。
```

**设计思路**：映射Dispatch的"前中期保护→后期摘除"设计。帮助玩家在前期稳定阵容方向，后期靠自己。适合放在银色海克斯，作为前期过渡辅助。

**海克斯：「触底反弹」（Bounce Back）**

```
类型：失败触发
效果：每次战斗失败后，下次商店刷新必定出现1个与你场上羁绊匹配的稀有度+1的棋子。
     效果不可叠加，胜利后移除。
```

**设计思路**：Dispatch的"失败重置保护"在MCGG商店系统中的直接映射。失败不是惩罚而是机会——给失败者一个翻盘契机。适合金色海克斯。

**海克斯：「绝望中的希望」（Hope in Despair）**

```
类型：全局被动
效果：所有技能/装备/海克斯的概率触发效果，触发率最低不少于20%。
     连续3次不触发后，第4次强制触发。
```

**设计思路**：将Dispatch的概率地板和窗口保护应用到技能触发上。让那些写30%实际体感10%的技能变得可用。适合金色海克斯，足以定义一个build方向。

### 棋子技能

**棋子：概率仲裁官（Probability Arbiter）**

```
费用：4费
定位：辅助/控制
技能：「命运校准」——选择场上一个友方棋子，使其下3次技能触发获得以下效果之一：
  - 若触发率>70%：必定触发
  - 若触发率<20%：触发率临时提升至25%
  - 连续2次未触发后：第3次必定触发
```

**设计思路**：将整个Dispatch系统浓缩到一个辅助棋子的技能中。4费保证了不会前期就凑到，但效果强大到可以围绕它构建阵容。"选择目标"的机制赋予玩家决策权——保护Carry还是保护前排？

**棋子：命运之镰（Scythe of Fate）**

```
费用：5费
定位：输出/收割
技能：「概率收割」——对目标造成伤害。
  被动：本英雄的攻击/技能触发率永远不低于20%；
  每次技能触发后，下次触发率-5%（最低回到原始值）；
  失败（技能未触发）后，触发率+10%（可超过原始值，无上限）
```

**设计思路**：5费核心C位，自带Dispatch式动态概率管理。触发后概率降低（避免太强），失败后概率上升（保证不会一直哑火）。这种动态调整让玩家在"触发→概率降→可能不触发→概率升→触发"的循环中感受命运的摆锤。

**棋子：善意之灵（Benevolent Spirit）**

```
费用：3费
定位：辅助/保护
技能：「庇护光环」——为周围友军施加保护：
  友军的技能触发率若低于20%，临时提升到20%（持续3秒）；
  每场战斗前2次失败的技能触发，自动改为成功。
```

**设计思路**：3费辅助，映射Dispatch的"概率地板+保护窗口"。前期可用，围绕它构建概率依赖型阵容（如装备了很多概率触发装备的阵容）。

## 总结

Dispatch的RNG均衡系统是近年来最优雅的概率设计之一。它的核心不是技术复杂，而是**设计哲学的转变**——从"概率是为了数学公平"到"概率是为了玩家感知公平"。五个子机制（76%阈值、3次窗口、失败重置、15%地板、终章摘除）形成了一个完整的情绪管理闭环。

对MCGG而言，这套系统最直接的应用场景是**PVE肉鸽模式的商店刷新**和**装备掉落**，其次是**技能触发概率的感知优化**。通过羁绊、海克斯和棋子技能三个维度，可以将这个设计哲学以不同粒度融入MCGG。

**核心迁移原则**：PVE场景可以大胆使用整套系统；PVP场景必须保持概率透明，最多使用双方均等应用的PRD或保底机制。
