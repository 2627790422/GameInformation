# 拆解结果：Joker触发链系统

- **来源游戏**：Balatro（LocalThunk / Playstack, 2024）
- **拆解时间**：2026-05-01
- **拆解者**：测试C（锅巴）
- **原始研究**：研究_20260501_Joker触发链系统.md

---

## 四层拆解

### 行为层

**玩家在游戏中的具体操作和行为模式：**

1. **收集与筛选**：玩家每回合从商店购买Joker牌，识别其类型（+Chips / +Mult / ×Mult / 条件触发）
2. **排列配置**：将Joker拖拽到最多5个槽位中，按预期触发顺序排列
3. **出牌验证**：选择手牌打出，观察得分计算过程，验证排列效果是否符合预期
4. **动态调整**：获得新Joker后，重新评估整体排列，反复尝试最优解
5. **牌型配合**：根据Joker链的特性调整出牌策略（走特定牌型路线以最大化触发收益）
6. **节奏管理**：在游戏中后期，决定是"保链"（不轻易更换成型Joker）还是"换链"（追求更大收益）

**行为特征总结：**
- 高频的**拖拽排序操作**——排序成本极低，鼓励反复实验
- **试错循环**——猜测排列 → 验证结果 → 调整排列
- 行为的策略密度远超传统卡牌构筑——每个Joker的"位置"和"角色"都需要思考

### 系统层

**支撑行为的规则、数据和底层机制：**

1. **固定触发顺序规则**：Joker效果严格按照玩家设定的从左到右顺序执行，前一个Joker的输出是后一个Joker的输入
2. **分阶段计算公式**：
   - 阶段1：基础筹码 + 基础倍率（由牌型决定）
   - 阶段2：Joker依次触发，每次修改筹码或倍率
   - 阶段3：最终得分 = 最终筹码 × 最终倍率
3. **效果类型体系**：
   - **筹码型（+Chips）**：加法，叠加到筹码池
   - **加法倍率型（+Mult）**：加法，叠加到倍率池
   - **乘法倍率型（×Mult）**：乘法，作用于当前倍率池
   - **条件型**：满足条件才触发（手牌花色、牌型、出牌数量等）
4. **槽位限制系统**：最多5个Joker，制造资源稀缺性和排列取舍压力
5. **反馈系统**：得分动画以链式方式逐个显示Joker触发效果，强化"顺序感"
6. **可视化辅助**：Joker卡面上清晰标注效果类型和数值，支持快速识别

**系统层核心洞察：**
- 系统的精妙处在于**将"策略配置"简化为一维线性排序**——把多维度的决策压缩成"左 vs 右"的二元选择，极大降低了认知负荷
- 加法优先乘法的数学原理是系统自带的"隐性引导"，优秀玩家会自然发现这个规律

### 决策层

**玩家在系统约束下做出的策略选择与权衡：**

1. **排列顺序决策**：同类Joker之间的顺序、不同类Joker之间的优先级
   - 关键公式：+筹码 → +倍率 → ×倍率（最优）
   - 何时打破规则？例如×Mult Joker在特定条件下收益更高时
2. **取舍决策**：在5个槽位限制下，哪些Joker保留、哪些替换
   - 短期收益 vs 长期成长：当前×2 vs 后期能成长到×5的Joker
   - 稳定性 vs 上限：稳定加法 vs 条件乘法（高风险高回报）
3. **路线决策**：根据Joker链的类型选择后续购买方向和牌型策略
   - 如果拿到多人头牌加成Joker → 走K/Q/J路线
   - 如果拿到花色加成Joker → 走同花路线
4. **风险决策**：是否保留经济追求理想Joker，还是立即强化现有链
5. **反制意识**：在PVP（或Balatro的自定义模式）中，预判对手可能的链设计针对性调整

**决策层核心洞察：**
- 玩家的决策颗粒度从"要不要这个棋子"细化到"放在哪个位置"，**策略深度翻倍但学习曲线平滑**
- 最优解不是固定的——随着游戏进程，Joker池变化迫使玩家持续重新评估

### 情感层

**玩家在过程中的情绪体验和心理变化：**

1. **探索期（前5回合）**：
   - 兴奋感：看到不同Joker组合的可能性
   - 困惑感：对排列顺序的"正确性"不确定
   - 轻微焦虑：怕放错顺序浪费了资源

2. **成长期（中期）**：
   - 成就感：发现一个高效组合时的"aha moment"
   - 掌控感：通过排列顺序精准控制得分
   - 期待感：每次升级商店都期待能拿到"补完链"的Joker

3. **巅峰体验**：
   - 心流状态：当Joker链完美运转、得分数字飙升时的高度沉浸
   - 惊喜感：「居然能这样算？！」——触发链的乘法效应超出预期时的震撼

4. **挫折与重试**：
   - 挫败感：精心排列的链被"条件不满足"打断时的失落
   - 不服感：「下一次一定更好」——驱动反复重试的动力
   - 社会比较欲：看到其他玩家更高得分时的挑战欲

5. **终局反馈**：
   - 满足感：最优链成型、打出超高分数时的宣泄
   - 回味感：想在下一局尝试另一种排列策略的探索欲

**情感层核心洞察：**
- 触发链的情感曲线是**U型 + 陡升**——前期轻度不适（困惑）→ 中期稳定成长（掌控）→ 后期爆发（心流 + 惊喜）
- "同一套牌，不同顺序 = 不同结果"这个特性创造了极强的**重玩欲望**

## MCGG迁移方案

### 羁绊效果：连锁之心（Chain Heart）

**全新羁绊——「连锁之心」**

**羁绊棋子池**（6枚棋子，1费~5费各一，再加1个5费核心）：

| 费用 | 棋子 | 定位 |
|------|------|------|
| 1费 | **链锤手**（战士） | 连锁起点：普攻施加"链印"标记 |
| 2费 | **传导者**（法师） | 链中：将"链印"传递给相邻友军 |
| 3费 | **聚合体**（坦克） | 链尾：每层"链印"提供减伤 |
| 4费 | **增幅核心**（辅助） | 链中：每经过一层"链印"，技能效果+20% |
| 5费 | **终末使者**（刺客） | 链尾：消耗所有"链印"，每层造成额外真实伤害 |
| 5费 | **链之女王**（法师） | 链中：回合开始将所有友军"链印"层数翻倍 |

**羁绊效果：**

- **2连锁**：所有拥有"链印"的棋子获得10%技能增幅
- **3连锁**：链式触发时，每经过一个友军，增益效果额外+15%
- **4连锁**：形成完整链时，触发"连锁爆发"——链末端棋子额外造成等同于链长(×60)的魔法伤害
- **5连锁**（链之女王在场时）：连锁爆发不消耗链印，且每回合首次触发时所有链印层数+1

**触发机制**：
- 战斗开始时，按棋子站位位置（从左到右）确定触发顺序
- 带有"连锁之心"羁绊的棋子序列中，前一个棋子技能结束后立即触发下一个棋子的技能（间隔0.3秒）
- 如果链中棋子阵亡，后续链中断（链印不会消失，但无"连锁"加成效果）
- 链末端棋子获得"连锁终结"标记，触发额外效果

### 海克斯Augment

**层级1（银色）：**

| 名称 | 效果 |
|------|------|
| **链式反应** | 连锁之心羁绊触发时，20%概率不消耗链印 |
| **排列艺术** | 你的棋子触发顺序可以指定为从右到左 |
| **链之起点** | 战斗开始时，最高星级的连锁棋子获得1层额外链印 |

**层级2（金色）：**

| 名称 | 效果 |
|------|------|
| **完美连锁** | 连锁之心羁绊从3连锁起生效（降低门槛） |
| **链式倍增** | 连锁触发的增益效果翻倍，但连锁最大长度-1 |
| **链中魔方** | 魔方格子允许你指定一个棋子为"链起点"（无视站位） |
| **传导增幅** | 链中棋子每传递一次链印，自身技能冷却-0.5秒 |

**层级3（棱彩）：**

| 名称 | 效果 |
|------|------|
| **无限连锁** | 连锁之心羁绊不再限制连锁长度上限 |
| **链之皇冠** | 获得一个2星「链之女王」。她的链印翻倍效果变为三倍 |
| **链的尽头** | 连锁终点棋子的技能会以伤害形式溢出到相邻对手棋子身上 |
| **超级连锁** | 所有连锁棋子基础属性+30%，但连锁中断时全队失去50%当前生命值 |

### 棋子技能

**1费 - 链锤手**（战士）
- **被动**：普攻对目标施加1层"链印"（持续整场战斗，不可驱散）
- **主动（40法力）**：挥动链锤，对前方扇形区域造成80/160/320物理伤害。每命中一个带"链印"的敌人，额外造成20/40/80伤害
- **链定位**：链起点——适合放在队列最左端

**2费 - 传导者**（法师）
- **被动**：回合开始时，将自身50%的链印层数传递给右边相邻友军
- **主动（60法力）**：释放传导波，链接目标敌人和两名随机敌人，造成120/240/480魔法伤害。链接生效期间，传导者每次获得链印时，被链接的敌人也受到等量伤害
- **链定位**：链中——链印传输枢纽

**3费 - 聚合体**（坦克）
- **被动**：每层链印提供8/16/32点双抗
- **主动（80法力）**：聚合链条，吸收周围所有友军的链印（最多5层），每吸收1层获得一个相当于最大生命值10%/15%/25%的护盾，持续4秒
- **链定位**：链尾——链印收益终端

**4费 - 增幅核心**（辅助）
- **被动**：每经过一次链触发，增幅核心的下一次技能伤害/治疗效果+25%（可叠加，上限200%）
- **主动（70法力）**：为核心友军注入增幅能量，使其接下来3秒内的技能伤害/治疗效果提升40%/60%/100%。如果增幅核心位于链中位置，此效果额外波及前后相邻的友军
- **链定位**：链中——链触发放大器

**5费 - 终末使者**（刺客）
- **被动**：战斗开始时，获得等同于当前链印层数×15/30/60的攻击力
- **主动（50法力）**：突进到链印层数最高的敌人身后，消耗目标身上所有链印，每层造成60/120/240真实伤害。如果此次击杀目标，刷新主动技能冷却
- **链定位**：链尾——链印消耗/终结者

**5费 - 链之女王**（法师）【核心】
- **被动**：回合开始时，所有携带链印的友军棋子链印层数翻倍
- **主动（100法力）**：女王领域——展开持续5秒的领域，领域内所有友军每次获得链印时额外获得1层。领域持续期间，连锁最大长度+2
- **链定位**：链中——链印放大器/链长扩展器

## 迁移适配要点

| Balatro原版 | MCGG适配 |
|-------------|----------|
| 5个Joker槽位 | 棋子站位（最多8格） |
| 左→右固定触发顺序 | 站位左→右触发顺序 |
| +Chips / +Mult / ×Mult | 链印层数作为"数值放大器" |
| Joker替换 | 棋子升星/替换 |
| 每回合验证排列 | 每场战斗验证排列 |
| 无可视化连线 | 战斗中显示链式触发连线动画 |
| 无直接反制 | 刺客可切后排打断链条 |

## 拆解总结

Joker触发链系统将"排列顺序"转化为核心策略维度，核心设计亮点是**用一维排序承载多维决策**——玩家只需要思考"左还是右"，但结果受筹码、倍率、条件触发等多种因素综合影响。MCGG迁移方案通过"链印"机制保留了Balatro的链式乘法思维，同时利用站位系统作为自然载体，新增「连锁之心」羁绊提供体系化支撑。低侵入方案（仅修改站位顺序语义）可以作为第一步快速验证，高侵入方案（完整羁绊+Augment）则可以作为新赛季的招牌玩法。
