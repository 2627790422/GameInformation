# 拆解结果：Samurai Bringer 阵型协同系统

## 基本信息
- **来源**：../研究/研究结果/研究_20260512_SamuraiBringer阵型协同自走棋.md
- **拆解时间**：2026-05-12 16:30 UTC
- **游戏**：Samurai Bringer: Rampage
- **机制**：阵型协同（Formation Synergies）— 武器类型+技能双层动态协同

---

## 四层拆解

### 行为层

**拿到阵型协同机制后，玩家行为变化**：

- 不再只盯着商店刷新凑种族/职业
- 改为**主动分配武器**给武士，构建协同链路
- 同样的武士阵容，换一套武器就变成完全不同的build
- 对战前会侦查敌方阵型，针对性调整武器分配
- 不再盲目追求阵容成型，而是**在速攻成型和后期爆发之间做路线选择**

**行为本质变化**：从「被动等待羁绊凑齐」变成「主动配置武器协同+战术路线选择」，玩家的核心操作从商店刷新迁移到了武器分配与阵型布局。

---

### 系统层

**这个机制改变了哪些底层系统**：

1. **羁绊触发模型重构**
   - 原本：羁绊由角色本身的种族/职业标签固定触发，触发条件刚性
   - 现在：羁绊由武器类型+技能匹配触发，触发条件可变、可配置
   - 本质：羁绊从"英雄属性"变成了"装备-英雄关系"，触发主动权交还给玩家

2. **资源分配维度扩展**
   - 原本：资源=经济（金币买英雄/升星），单一维度
   - 现在：资源=经济+武器分配（哪个武士拿什么武器），双维度并行
   - 本质：武器不再只是数值增幅器，而成为策略构建的核心资源

3. **节奏控制系统引入**
   - 原本：自走棋天然偏向中期发力，节奏由商店刷新频率决定
   - 现在：玩家可选择速攻路线（低费武器快速触发协同）或后期路线（积累高费武器厚积薄发）
   - 本质：在自走棋框架中引入了"节奏选择"这一元策略层

---

### 决策层

**短期决策（战斗准备中的即时选择）**：
- 这把给哪个武士装什么武器？是凑武器类型协同还是追求个体属性最优？
- 看到敌方阵型后，要换武器反制还是坚持自己的build？
- 武器不够所有人用，优先武装核心Carry还是先触发2人小型协同？

**长期决策（构筑方向）**：
- 开局将军选择即锁定策略框架——速攻型将军选低费武器路线，后期型将军优先攒资源
- 招募武士时不再只看个体强度，要看「这个武士能搭配什么武器类型」
- 武器获取路线决定后续所有装备决策——拿了刀系武器就偏向刀系武士，拿了弓系就优先弓手
- 双策略路线分化意味着：选择了速攻就放弃后期翻盘能力，选择了后期就要扛住前中期压力

---

### 情感层

**爽点分析**：

1. **掌控感的爽**
   - 羁绊不再是"等发牌员给"，而是"我自己配出来"
   - 同样的英雄阵容可以通过武器分配打出完全不同的效果
   - 玩家感受到：这是我的build，不是系统随机给的

2. **解谜成功的爽**
   - 侦查敌方阵型后针对性反制，有一种"我看穿你了"的智力快感
   - 武器+技能的双层协同叠加，像在解一个多变量优化谜题
   - 找到最优武器分配方案时的满足感

3. **路线博弈的爽**
   - 速攻 vs 后期的选择本身就是博弈——不是单纯数值对抗，而是时间节奏对抗
   - 速攻成型压制对手后快速结束战斗的碾压感
   - 熬过前期劣势、后期一波翻盘的逆袭感

4. **异步征服的爽**
   - 打败全球玩家的阵型，且对方还不知道自己被打了
   - 每次对战都是"一个人的智力对抗另一个人的智力"，没有操作干扰

**情绪类型**：掌控+解谜+节奏博弈

## 玩法块公式

| 维度 | 块公式 |
|------|--------|
| **羁绊触发** | 原本羁绊由角色种族/职业标签被动触发，现在由武器类型+技能匹配主动配置触发 |
| **资源策略** | 原本队伍构建围绕单一经济维度（金币→英雄→升星），现在武器分配成为与经济并列的第二策略维度 |
| **节奏选择** | 原本自走棋节奏由商店刷新和升星节奏决定，现在玩家在战前即可选择速攻/后期两条不同的时间策略路线 |
| **对战模式** | 原本对战是实时匹配+即时操作，现在对战是异步对阵+纯策略预判（阵型保存→他人挑战→AI代为执行） |
| **侦查反制** | 原本PvP信息不对称靠猜测对手build，现在可查看敌方阵型后针对性调整，形成石头剪刀布元游戏 |

## MCGG迁移方案

### 羁绊效果

**装备协同羁绊（Equipment Synergy Trait）**

```
「武器共鸣」羁绊（2/4/6）
- (2) 场上每有2个装备相同武器类型的英雄，这些英雄获得15%攻击力加成
- (4) 加成提升至30%，并且武器类型协同的英雄获得10%伤害减免
- (6) 加成提升至50%，武器类型协同的英雄攻击时有20%几率触发武器共鸣，造成额外伤害
```

**设计思路**：将SBR的武器类型协同映射为MCGG的装备类羁绊。不再是英雄本身的种族/职业，而是「装备了什么武器」决定协同。这鼓励玩家在商店中不仅考虑英雄组合，还要考虑武器规划。

### 海克斯/Augment

**① 战术姿态（Tactical Stance）— 白银海克斯**

```
【速攻姿态】前3个回合所有友方英雄获得25%攻速加成；第5回合开始该加成每回合衰减10%直至消失
【坚守姿态】前3个回合所有友方英雄伤害降低20%；第5回合开始每回合获得8%伤害加成（可叠加至40%）
→ 玩家在对战开始前2选1，不可更改
```

**设计思路**：直接映射SBR的速攻/后期路线分化。开局选姿态决定整局节奏，增加战前策略选择。

**② 武器精通（Weapon Mastery）— 黄金海克斯**

```
选择一个武器类型（剑/弓/杖）：
该类型武器在战斗中效果翻倍；但其他类型武器效果减半
```

**设计思路**：鼓励玩家聚焦单一武器类型构建，类似SBR的武器协同深度。

**③ 阵型侦查（Formation Scout）— 棱彩海克斯**

```
每回合开始时可查看下一个对手的阵容配置（英雄、装备）
基于此信息，你有10秒时间调整你的站位和装备分配
```

**设计思路**：保留SBR的侦查-反制元游戏精髓，但限制为下回合可调整而非实时反制。

### 棋子技能

**武器大师·千兵卫（Weapon Master）— 5费传说**

```
【武器切换】
被动：战斗开始时，根据装备的武器类型获得不同技能效果
- 剑：技能变为「居合斩」— 对前方3格造成大量物理伤害
- 弓：技能变为「穿云箭」— 对最远敌人造成伤害并击退
- 杖：技能变为「结界」— 为周围友军提供护盾
- 刀：技能变为「乱舞」— 对周围所有敌人造成多段伤害
```

**设计思路**：一个英雄多种技能，完全取决于装备什么武器。体现了SBR"武器改变能力"的核心设计。这个英雄会成为阵容构建的灵活支点——根据对手类型决定给什么武器。

## 通用设计启示

1. **羁绊不一定是角色的**：装备、站位、甚至战斗节奏都可以成为羁绊触发维度
2. **给玩家控制权**：把触发条件从被动（凑角色）变为主动（分配装备），玩家的参与感完全不同
3. **节奏选择是元策略**：速攻vs后期的二分框架简单但有效，适合自走棋这种以回合为单位的游戏
4. **异步PvP降低门槛**：不要求实时匹配，每个玩家都是"阵型创作者"，增加PvP可玩性而不增加等待成本
