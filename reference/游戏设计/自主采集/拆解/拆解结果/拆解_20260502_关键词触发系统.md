# 拆解结果：关键词触发系统

**来源**：王者万象棋（腾讯）
**机制名称**：关键词触发系统（Keyword Trigger System）
**拆解时间**：2026-05-02
**执行者**：芙芙
**MCGG适配度**：0.85

---

## 基本信息

- **研究来源**：`../研究/研究结果/研究_20260430_2026自走棋新机制研究.md#机制2`
- **机制类型**：羁绊替代型（Synergy Replacement）
- **核心变化**：从"凑羁绊人数"→"组合词条效果"

---

## 四层拆解

### 行为层

**核心变化**：从"被动的羁绊凑数"→"主动的效果编排"

| 维度 | 传统羁绊 | 关键词触发系统 |
|------|---------|---------------|
| 阵容构建 | "我有3个战士，再找一个开4战士" | "这个词条组合能触发什么效果？" |
| 棋子价值 | 棋子 = 职业+种族标签 | 棋子 = 效果触发器 |
| 玩家行为 | 凑数：被动等待第N个同羁绊棋子 | 编排：主动组合词条触发链 |
| 学习曲线 | 线性（记住羁绊表） | 网状（探索词条组合） |

**关键行为转变**：玩家从"拼图者"（fitting pieces）变为"编曲者"（composing effects）。阵容不再是一张静态表格，而是一套动态效果链。

---

### 系统层

#### 系统1：词条类型设计

| 词条类型 | 触发时机 | 设计意图 |
|---------|---------|---------|
| 登场 | 棋子放置/上场时 | 即时收益，鼓励换阵 |
| 牺牲 | 棋子死亡时 | 逆风翻盘，赋予死亡价值 |
| 退场 | 棋子退场/售出时 | 经济-战斗转化 |
| 闪现 | 跨位置移动时 | 战术位移，空间博弈 |
| 持续 | 每回合持续生效 | 稳定增益，长线投资 |
| 爆发 | 一次性触发 | 关键回合爆发 |

**设计意图**：词条覆盖战斗全生命周期（登场→持续→爆发→牺牲），让每个时刻都有策略意义。

#### 系统2：词条组合规则

| 规则 | 说明 |
|------|------|
| 每棋子1-2个词条 | 控制复杂度 |
| 同类型词条可叠加 | "牺牲"+"牺牲"=双倍效果 |
| 不同类型可联动 | "登场"+"持续"=登场后持续增益 |
| 稀有词条限1 | 防止过度堆叠 |

#### 系统3：与传统羁绊的共存

**不取代，而是扩展**：
- 羁绊提供基础数值框架（+攻击/+生命）
- 词条提供效果变体（触发型机制）
- 两者叠加产生"羁绊×词条"的组合深度

---

### 决策层

#### 短期决策（每局）

- **词条组合选择**：这局走"牺牲流"还是"登场流"？
- **棋子替换时机**：是否为了触发"退场"效果故意卖出棋子？
- **站位调整**：是否为了触发"闪现"效果而调整阵型？
- **资源分配**：经济用于D卡凑词条，还是升级人口？

#### 长期决策（跨局/赛季）

- **词条流派专精**：专精某类词条组合
- **反制策略**：如何针对流行词条流派
- **赛季词条变化**：赛季轮换词条的选择策略

#### 决策复杂度评估

| 维度 | 传统羁绊 | 词条系统 |
|------|---------|---------|
| 决策点/回合 | 2-3个 | 3-5个 |
| 组合空间 | N个羁绊 | N×M个词条组合 |
| 信息透明度 | 高（羁绊表可见） | 中（组合效果需探索） |

---

### 情感层

#### 爽点分析

| 爽点 | 为什么爽 |
|------|---------|
| **组合发现的爽** | 发现"牺牲+治疗"的词条联动时，"原来还能这样！"的恍然大悟 |
| **效果链触发的爽** | "登场→爆发→牺牲"连续触发时，屏幕像放烟花 |
| **逆风翻盘的爽** | 死亡触发的牺牲词条在劣势时产生巨大逆转 |
| **个人风格的爽** | 我的词条组合独一无二，不是抄攻略的标准化阵容 |

#### 情绪类型

- **好奇 (Curiosity)**：探索词条组合的过程
- **惊喜 (Surprise)**：意外的词条联动效果
- **掌控 (Mastery)**：精通词条系统后的自信
- **独创 (Originality)**：创造独有流派的自豪

#### 潜在风险

- **信息过载**：新玩家面对20+词条可能感到困惑
- **平衡难度**：词条组合容易产生OP（过强）或UP（过弱）流派
- **抄作业**：如果攻略站列出"最优词条组合"，系统会退化为新羁绊表

## MCGG迁移方案

### 词条设计（10个核心词条）

| 词条 | 效果 | 稀有度 |
|------|------|--------|
| 登场·冲击 | 登场时对最近敌人造成伤害 | 普通 |
| 登场·守护 | 登场时为相邻友军提供护盾 | 普通 |
| 牺牲·献祭 | 死亡时为全体回复生命 | 稀有 |
| 牺牲·复仇 | 死亡时对击杀者造成伤害 | 普通 |
| 退场·回响 | 售出时返还50%金币 | 稀有 |
| 闪现·突袭 | 跨行移动时下次攻击暴击 | 普通 |
| 持续·增幅 | 存活时每回合叠加伤害 | 稀有 |
| 爆发·终幕 | 唯一一次释放终极技能 | 史诗 |
| 联动·共振 | 相邻友军触发词条时自己也触发 | 史诗 |
| 觉醒·进化 | 3星时词条效果翻倍 | 稀有 |

### 羁绊层面

**建议新羁绊：词条师（Wordsmith）**

| 人数 | 效果 |
|------|------|
| 2 | 词条触发时额外产生1次效果 |
| 3 | 词条触发效果提升50% |
| 4 | 词条可触发2次（CD减半） |

### 海克斯

- **词条共鸣（Gold）**：本局获得的第一个词条免费复制给另一个棋子
- **词条大师（Prismatic）**：所有棋子获得随机稀有词条
- **词条编辑（Silver）**：可以移除并替换一个棋子的词条

## 设计总结

| 维度 | 核心洞察 |
|------|---------|
| **行为层** | 词条系统将阵容构建从"凑数"升级为"编排"，提升策略上限 |
| **系统层** | 词条作为独立效果层，与羁绊数值层并行，形成双层策略空间 |
| **决策层** | 词条组合增加每回合的有意义决策点，从2-3个增至3-5个 |
| **情感层** | 核心爽点在于"发现组合"和"独创流派"，而非数值碾压 |
