## 四层拆解

### 行为层

拿到这个机制后，玩家的行为发生了哪些改变：

| 传统TFT PvP | 敖兴崛起PvE |
|-------------|------------|
| 不再需要关注其他7家阵容 | 只需关注自身build和敌人Modifier |
| 不再被动接受固定的难度曲线 | 改为主动选择每一段路的难度参数 |
| 不再被回合倒计时压迫 | 可以花任意时间分析决策 |
| 不再输了就出局 | 改为失败了获得补偿+从头再来 |
| 不再每局从零开始 | 改为跨局积累元进度解锁新选项 |
| 不再"这次运气好才能赢" | 改为"这次选对了debuff组合就能赢" |

**行为本质变化**：从"对抗人类对手的竞技博弈"变成"构建自己挑战路径的策略工程"——玩家从被动适应者变成主动设计师。

---

### 系统层

这个机制改了哪些底层规则（系统层是核心层）：

#### 1. 难度生成系统
- **原本**：对手的阵容/装备/经济水平决定你的难度（PvP匹配）
- **变为**：玩家通过Celestial Challenges主动选择敌方增益，通过Boss Modifier选择规则变异——难度是玩家自己"组装"出来的

#### 2. 失败处理系统
- **原本**：输掉关键回合 = 出局，游戏结束，挫败感强
- **变为**：生命值系统 + 失败补偿掉落（金币/装备/英雄复制器）— 失败变成"资源重置+再来一次但更强"的肉鸽循环，而非惩罚

#### 3. 元进度系统
- **原本**：每局独立，不携带任何跨局资源或能力
- **变为**：Journey Track（80级）无论胜负都积累——但核心不是数值加成（HP+10%），而是解锁新玩法选项（新向导、向导变体），拓展策略多样性

#### 4. 时间压力系统
- **原本**：PvP回合倒计时强制快速决策，时间压力是难度的一环
- **变为**：完全取消时间限制，玩家可以慢速分析Celestial Challenge×Boss Modifier×阵容的交互——降低焦虑、提升策略深度

#### 5. 辅助系统
- **原本**：没有外部辅助，全靠玩家自己的知识和操作
- **变为**：Guide Masters提供战斗辅助（Pantheon天降AOE、Soraka回血）或经济辅助（Zoe随机掉落、Teemo提前5费），形成战斗/经济双轨分流

---

### 决策层

这个机制如何改变玩家的决策思路：

#### 短期决策（战斗中的即时选择）

1. **Celestial Challenge选择**：不是盲选高金币——必须评估当前阵容弱点。后排carry阵容不能选"敌方前排跳后排"的debuff，爆发阵容不怕"敌方坦克化"的debuff
2. **Boss Modifier选择**：不是选"看起来最弱的Boss"——要考量Modifier对当前build的影响。Viego的击杀回血对消耗战阵容是灾难，Darius的死亡爆炸对近战阵容是噩梦
3. **Boss Modifier × Celestial Challenge组合评估**：某些Modifier会放大特定Challenge的危险性——需要预判合成效应
4. **站位决策**：针对Boss特定技能调整，而非通用的对手counter站位

#### 长期决策（build构筑方向）

1. **Guide Master选择改变整个build方向**：选Zoe（经济辅助）→走快速成型路线，多刷装备；选Soraka（战斗辅助）→走持久战路线，注重前排坦度；选Teemo（经济辅助）→走赌狗路线，移除3/4费专注低费三星
2. **Celestial Challenge的累积效应改变后续pick逻辑**：如果前期选了"敌方加攻速"的debuff，后期就不该走需要拖时间的阵容
3. **元进度规划**：解锁哪些向导变体？用哪些向导登顶（皮肤成就）？——跨局决策影响单局build方向
4. **风险预算**：生命值系统意味着可以承受几次失败——这改变了"是否all-in"的决策阈值

### 情感层

玩家用这个机制时的爽感分析：

#### 爽点列表

1. **主动权握在手里的爽感**
   - 为什么爽：传统肉鸽的难度是系统决定的，这里是"我"决定的——"这把难是因为我贪，不是系统恶心我"
   - 场景：选了最难debuff拿到最高金币，然后在装备碾压下横扫

2. **算对了的智力满足**
   - 为什么爽：Celestial Challenge × Boss Modifier × 阵容的三重交互，当你预判到组合效应并选对了，那种"我懂了这游戏的底层逻辑"的感觉
   - 场景：Boss Modifier是"敌方死亡爆炸"，你选了"敌方近战减伤"的Celestial Challenge（因为你的阵容是全远程）

3. **风险筹码的刺激感**
   - 为什么爽：每个Celestial Challenge都是一次"下注"——"我觉得我的阵容能扛住这个"
   - 场景：选了最危险的金币最高选项，结果差一丝血没扛住——但"下次我一定行的"

4. **失败被温柔对待的安全感**
   - 为什么爽：传统自走棋输关键局直接出局=20分钟白费；这里输了有补偿="没关系的，再来一次会更强"
   - 场景：Boss战翻车，但获得一个英雄复制器+金币，重整旗鼓下一区更强

5. **没有时间压力的从容感**
   - 为什么爽：可以慢悠悠地分析每个选项的利弊——像下棋一样思考，而不是像打游戏一样拼手速
   - 场景：在Celestial Challenge界面花2分钟仔细评估每个debuff对阵容的影响

6. **收集养成的渐进快感**
   - 为什么爽：解锁新向导→新build可能性→新登顶体验——形成一个"玩法多样性不断扩大"的循环
   - 场景：用第12个不同向导登顶，拿到Portal皮肤的那一刻

#### 情绪类型

**核心情绪**：自主掌控 + 智力满足 + 风险刺激
**辅助情绪**：安全感 + 从容感 + 收集养成

## 抽象玩法块公式

### 核心公式

> **"原本难度被动递增，现在玩家主动选择如何变难"**

### 子公式

| # | 公式 | 说明 |
|---|------|------|
| 1 | 原本对手决定难度 → 现在玩家选择敌人增益换经济补偿 | 难度来源从"外部竞争"变为"内部选择" |
| 2 | 原本失败即出局 → 现在失败获补偿+重试更强 | 失败从"终点"变为"循环节点" |
| 3 | 原本单局重置 → 现在跨局解锁新策略选项 | 元进度从"变强"变为"变多样" |
| 4 | 原本时间压力驱动决策 → 现在无限制慢速分析 | 压力源从"时间"变为"信息复杂度" |
| 5 | 原本孤立build构筑 → 现在外部辅助重塑build路径 | 向导系统创造战斗/经济双轨策略空间 |

## MCGG迁移方案

### 羁绊效果

#### 羁绊1：命运契约（3/5）

- **(3)** 每回合开始时，从2个敌方增益中选择1个应用给敌方全队。根据选择的增益难度，获得1-3金币。
- **(5)** 可选择3个增益，并且获得的金币翻倍。此外，每选择3次增益后，全队获得一层永久护盾（+200HP）。

**设计依据**：Celestial Challenge的"主动选择debuff换资源"核心逻辑，适配MCGG羁绊体系。

**适配度**：0.85（可直接迁移，MCGG已有回合开始触发机制）

#### 羁绊2：败者复仇（2/4）

- **(2)** 败者复仇棋子阵亡时，为场上随机友军回复20%最大生命值。
- **(4)** 每次输掉回合后，下一个回合全体棋子获得+30%攻击力和+30%法强（可叠加2次）。

**设计依据**：敖兴崛起的"失败补偿+再来一次更强"逻辑。

**适配度**：0.75（需要改造——MCGG的失败判定和补偿机制需适配）

### 海克斯/Augment

#### 海克斯1：命运抉择（黄金海克斯）

- **效果**：从下一阶段开始，每阶段初弹出一个"命运抉择"界面，从3个敌方增益中选择1个。选择的增益持续整个阶段，同时立即获得对应的奖励（金币/装备/英雄）。
- **增益示例**：
  - 绿色：敌方全体+10%攻击力 → 奖励：3金币
  - 蓝色：敌方额外召唤2个小怪 → 奖励：1件随机蓝装
  - 紫色：敌方获得一个随机的3星1费单位 → 奖励：1件随机紫装

**设计依据**：Celestial Challenges的核心循环。

**适配度**：0.70（需要改造——MCGG需要实现阶段初弹窗UI）

#### 海克斯2：向导之助（白银海克斯）

- **效果**：选择一个"向导"效果：
  - **战斗型**：开战时召唤一个不可选中的辅助单位，对敌方后排造成一次AOE伤害（Pantheon式天降）
  - **经济型**：每回合商店额外刷新1次，且第3次刷新必定出现比当前等级高1费的棋子（Zoe式掉落）

**设计依据**：Guide Masters的战斗/经济双轨分流。

**适配度**：0.80（可直接迁移——MCGG海克斯系统已有类似功能）

#### 海克斯3：重振旗鼓（棱彩海克斯）

- **效果**：每次输掉回合后，获得1个英雄复制器（最多3次）。此外，输掉回合后，全队棋子永久+100HP（无上限）。

**设计依据**：敖兴崛起的"失败补偿+肉鸽循环"逻辑。

**适配度**：0.65（需要改造——MCGG中的英雄复制器是珍贵资源，需要平衡调整）

---

### 棋子技能

#### 棋子1：Celestial Guide（5费传说）

- **技能 - 天界抉择**：在战斗开始前，选择一项效果：
  - **战斗祝福**：本回合友方全体获得400/600/1000护盾，持续8秒
  - **经济祝福**：如果本回合获胜，额外获得2/3/5金币
- **羁绊**：命运契约 + 辅助

**设计依据**：Guide Master的双轨选择设计，转化为棋子技能。

**适配度**：0.75（需要在战斗前弹窗选择，MCGG需要UI适配）

#### 棋子2：The Challenger（4费史诗）

- **技能 - 风险契约**：对敌方全体施加一个debuff（降低15/25/40护甲魔抗），持续整场战斗。但如果本回合战败，该棋子永久损失200最大生命值。
- **被动**：每存活赢下3个回合，该棋子永久+300最大生命值。

**设计依据**：Celestial Challenge的"风险-回报"核心循环。

**适配度**：0.70（永久生命值变动在MCGG中较少见，需要验证平衡性）

#### 棋子3：Phoenix Reborn（3费稀有）

- **技能 - 涅槃重塑**：阵亡后3秒原地复活，复活后获得一个随机装备（每个装备槽独立计算，最多3次）。每次复活后，装备品质提升一个等级（白→蓝→紫）。

**设计依据**：敖兴崛起的"失败→补偿→更强"肉鸽循环。

**适配度**：0.82（可直接迁移——死亡触发+装备获取是MCGG已有机制）

| 优先级 | 方案 | 理由 |
