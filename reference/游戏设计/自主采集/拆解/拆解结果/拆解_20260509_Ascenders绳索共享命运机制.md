## 四层拆解

### 行为层

拿到绳索共享命运机制后，玩家行为发生以下变化：

| 行为变化 | 原本 | 现在 |
|----------|------|------|
| 棋子站位 | 独立计算最优位置 | 必须考虑"绳索拓扑"——谁在上方谁在下方 |
| 受伤承担 | 每个棋子独立承受伤害 | 伤害可分担、护盾可共享、一人失误全员遭殃 |
| 角色牺牲 | 棋子死亡=减员，被动接受 | 可选择"割绳"主动牺牲一个棋子，换取其余棋子爆发 |
| 阵容构筑 | 只看棋子个人能力 | 还要看"连接适配性"——高低位搭配、坦承比例 |
| 战斗节奏 | 每轮结束后独立判定 | 引入"救援点数"资源，可在战斗中复活链接棋子 |
| 风险感知 | 局部风险（谁在扛伤害） | 全局风险（链上任何一环断了都会影响全链） |

**行为本质变化**：从"独立棋子各自为战"变成"链接棋子的命运共同体管理——每个站位决策都影响整条链的生死"。

---

### 系统层

这个机制改变了以下底层系统：

1. **单位关系模型重构**
   - 原本：棋子间仅通过"数值加成"关联（羁绊攻速+20%、攻击力+30%等）
   - 现在：棋子间建立"物理约束"——连接不是数字buff，而是一个真实的约束条件，改变了输出模型、生存模型和站位逻辑

2. **风险分配机制转变**
   - 原本：风险独立——脆皮被秒只影响一个棋子
   - 现在：风险绑定——链上任何一点的失误都会传导到整条链，但同样链上的任何增益也会共享。从"独立风险"转为"分布式风险"

3. **战术资源引入：救援点数 + 割绳决策**
   - 原本：战斗中的战术资源主要是法力值、血量、装备触发
   - 现在：新增两个元资源——
     - **救援点数**：有限次数复活链接棋子（类似"续命"概念但只对链接棋子有效）
     - **割绳**：不可逆的牺牲决策，用永久失去链接换短期爆发

---

### 决策层

**短期决策（战斗中）**：
- 站位拓扑：哪个棋子放高位（暴露但输出），哪个放低位（安全但被动）？
- 救援时机：队友坠落时，消耗宝贵的救援点数还是接受减员？
- 割绳时机：血量危机时是否割绳——用1个棋子的永久失去换其余棋子的爆发+无敌？

**长期决策（阵容/构筑）**：
- 链接棋子选择不再只看"谁强"，还要看"谁和谁适合绑一起"——高费大核搭配低费工具人创造非对称价值
- 肉盾+脆皮链接会改变出装逻辑：脆皮不再需要纯防装，可以由肉盾分担伤害
- "牺牲流"构筑成为可能——故意链接一个低费棋子，关键时刻割绳让高费棋子爆发
- 控场棋子（眩晕/减速）放在链接高位可以保护低位队友——位置与功能性产生新的化学反应

> 决策层核心改变：连接关系从"组完阵容就固定"变成"需要在战斗中持续管理的动态关系"。

---

### 情感层

1. **规则破坏的爽感**
   - "我不用常规阵容，我用链接把两个看似不搭的棋子绑定，产生奇异效果"
   - 羁绊从"凑数值"变成"创造新关系"的满足感

2. **高爆发牺牲的戏剧张力**
   - 割绳瞬间：「对不起兄弟，但我需要这3秒无敌秒对面后排」
   - 主动牺牲不是惩罚，是战术选择——这是"我做了残忍但正确的决定"的控制感

3. **连锁反应的紧张刺激**
   - 一人失误全员遭殃 → 每一步站位都在赌命
   - "下面那个能不能扛住……扛不住我就没了"的持续心跳

4. **救援成功的成就感**
   - 耗尽最后1个救援点数把核心棋子拉回来 → 逆转的快感
   - 类似MOBA中"极限救队友"但发生在自走棋框架内

5. **命运共同体的叙事感**
   - 两个棋子"同生共死"产生情感羁绊——玩家会记住"那把链接的龙龟和拉克丝"

**情绪类型**：规则破坏 + 高风险高回报 + 戏剧性牺牲 + 逆袭快感

## 玩法块抽象公式

| 层 | 公式 |
|----|------|
| **核心公式** | 原本棋子间只有数值加成关系，现在棋子间建立**物理化连接**，共享命运（伤害分担/护盾共享/连锁死亡），并可通过**主动牺牲一个**换取其余棋子的短期爆发 |
| **风险公式** | 原本风险独立（每个棋子自己扛），现在**风险分布式传导**（链上任何一点受伤，全链受影响，但链上任何增益也共享） |
| **取舍公式** | 原本棋子死亡=被动减员，现在**割绳=主动牺牲**——用永久失去链接换3秒无敌+30%攻击力 |
| **构筑公式** | 原本只看棋子个人强度和羁绊数值，现在还要看**链接适配性**（高位输出型+低位保护型搭配） |

## MCGG迁移方案

### 羁绊效果

**「命运」羁绊**（推荐首发方案）

```
触发条件：激活特定羁绊（如"命运"羁绊）

Lv1（2棋子）：
  链接的棋子在战斗中共享护盾值——一个棋子获得护盾时，链接棋子也获得50%

Lv2（3棋子）：
  链接棋子之一死亡时，若链接棋子存活，消耗"救援点数"将其复活（每局上限2次）
  救援点数：每局初始2点，不可恢复

Lv3（4棋子）：
  链接棋子共享伤害减免——一人受到伤害时，链上其他人分担15%
  分担方获得"减伤"处理（不直接扣血），避免脆皮被连带秒杀

惩罚机制（核心差异点）：
  当链接棋子之一血量低于20%时，玩家可选择"割断链接"：
  - 割断棋子立即死亡
  - 剩余链接棋子获得3秒攻击力+30% 且 无法被选中
  - 该棋子永久失去链接效果（不可逆）
  每局最多割断1次
```

**适配度**：0.79 | **迁移类型**：可改造后迁移

### 海克斯/Augment

**海克斯：命运之绳**

```
效果：选择2个棋子，在战斗中通过"命运之绳"连接。
  - 链接棋子共享15%所受伤害（分担方减伤处理）
  - 链接棋子之一死亡时，另一个获得+25%攻击力（复仇），持续5秒
  - 每个棋子最多1个链接
类型：金色海克斯
获取条件：任意阶段
```

**海克斯：牺牲协议**

```
效果：你可以选择一个棋子标记为"牺牲者"。
  牺牲者死亡时，其他同羁绊棋子获得：
  - 3秒无法被选中
  - +35%攻击力
  - 回复20%最大生命值
类型：棱彩海克斯
限制：每局仅触发1次
```

**海克斯：命运共同体**

```
效果：随机指定你阵容中的3个棋子建立链接。
  链接效果：共享治疗（一人获得治疗，其余2人获得40%）、共享控制免疫（一人免控，其余也免控）
  弊端：一人被眩晕时，其余2人减速30%
类型：金色海克斯
```

### 棋子技能

**棋子：命运编织者（5费）**

```
技能 - 命运链接：
  选择最近的友方棋子建立链接（持续整场战斗）
  链接棋子共享：
  - 25%护盾
  - 15%伤害分担
  当链接棋子死亡时，命运编织者获得+40%法术强度（持续至战斗结束）
  链接不可切换，每场战斗只能链接1个棋子
```

**棋子：绳索大师（3费）**

```
技能 - 救援钩：
  向血量最低的友方棋子投掷钩锁
  - 若目标存活：为其回复15%最大生命值
  - 若目标已死亡：以30%血量复活（每局最多1次）
  - 自身获得目标死亡前最后受到伤害量50%的护盾（复仇护盾）
```

### 关键平衡参数

| 参数 | 建议初始值 | 调参方向 |
|------|-----------|---------|
| 护盾共享比例 | 50% | 太强→30%，太弱→70% |
| 救援点数上限 | 2次/局 | 核心平衡点 |
| 伤害分担比例 | 15% | 分担方减伤处理 |
| 割绳后攻击力加成 | +30% | 值得割但不无脑割 |
| 割绳后无敌时间 | 3秒 | 太短无战术价值，太长无脑 |
| 链接人数上限 | 4人(Lv3) | 控制连锁规模 |
