## 四层拆解

### 第一层：行为层

**问题**：拿到这个机制后，玩家的行为发生了什么改变？

#### 行为变化

| 维度 | 传统自走棋 | Hades-like 模式 |
|------|-----------|----------------|
| 对局心态 | 赢了爽/输了烦 | 赢了推进战斗力，输了推进故事 |
| 返回基地 | 纯功能（排队/选模式） | 核心循环节点（看对话/推进关系/解锁内容） |
| 棋子关系 | 挑选战斗工具 | 培养感情、挖掘角色故事 |
| 失败处理 | 掉分→受挫→可能弃坑 | 触发新叙事→期待下次→继续游玩 |
| Run间动机 | "再打一把冲分" | "回去看看谁在等我" |
| 内容消耗 | 追新棋子/皮肤，用完即弃 | 深挖已有角色故事，情感绑定 |

#### 行为本质变化

**从"逐局竞技→结算→再匹配"的线性循环，变成"每局都是故事旅行的下一站，胜负只是分叉方向不同"的螺旋循环。**

玩家不再把每局当作孤立的胜负事件，而是当作一段持续叙事旅程中的一站——输赢都推进不同维度的进度。

---

### 第二层：系统层

**问题**：这个机制改了哪些底层规则？

#### 系统改变1：游戏循环重构

| 维度 | 原本 | 变为 |
|------|------|------|
| 循环结构 | 匹配→战斗→结算→再匹配（线性） | 战斗→基地叙事枢纽→能力成长→再战斗（螺旋） |
| 返回基地的意义 | 暂停/等待（被动节点） | 核心内容消费时刻（主动节点） |
| 进度维度 | 单轨（RANK/段位） | 双轨并行（战斗力 + 叙事） |

#### 系统改变2：奖励模型转换

- **原本**：掉落驱动 — 打败怪物→获得物品→装备使用
- **变为**：角色关系驱动 — 打败怪物→遇见角色→角色给奖励→关系加深→更多内容解锁

**关键**：角色不仅是功能入口（给Boon），也是叙事锚点——功能和故事绑定，让每次奖励获取都附带情感价值。

#### 系统改变3：失败系统重定义

- **原本**：失败 = 负反馈终结点（Game Over / 掉段 / 惩罚）
- **变为**：失败 = 叙事分叉节点（返回基地→触发新对话→解锁新线索→积累永久进度）

**架构核心**：叙事进度和战力进度**解耦**。败局影响战力轨道，但不影响甚至推进叙事轨道。这消除了roguelike/竞技中最大的痛点：白忙一场。

#### 系统改变4：Hub-as-Progress（枢纽即进度）

- **原本**：基地 = 功能菜单（开始游戏/商店/设置）
- **变为**：基地 = 叙事空间 + 社交互动 + 永久成长中心

基地不再是一个"等待界面"，而是一个"内容消费界面"——玩家回到基地不是为了等待，而是为了体验。

### 第三层：决策层

**问题**：这个机制如何改变玩家的决策思路？

#### 短期决策

- **棋子选择不再只看战力**：选棋子时需要考虑"这个棋子的故事线我推进到哪了？再选它会不会触发新对话？"
- **败局时的心态切换**：输掉一局后不再是"亏了"，而是"回去看看有没有新叙事触发"——改变了挫败后的行为链
- **资源分配的叙事维度**：要不要花金币养某个棋子的关系值，还是强化当前阵容？多了一个资源博弈层

#### 长期决策

- **故事线深挖vs广度探索**：选择深耕某个角色的故事线（高羁绊值→专属技能→外观）还是广度发展多个角色？这改变了棋子使用分布
- **赛季目标的叙事化**：不只是冲段位，还要完成角色故事——为竞技疲惫的玩家提供了第二目标
- **棋子池的"情感绑定"**：当你对某个棋子有情感投入，即使它被削成下水道你也会继续用——这改变了Meta驱动的棋子选择逻辑

### 第四层：情感层

**问题**：玩家用这个机制时的爽感是什么？

#### 爽点1：失败被"救赎"的叙事爽感

败局不再是纯粹的负反馈，而是"解锁新剧情的机会"。玩家心理从"又要掉分了，好烦"变成"虽然输了，但回去看看有没有新对话"。这种"失败有意义"的底层体验，是Hades-like最核心的情感魔法。

#### 爽点2：角色牵绊的温暖感

战斗结束后回到基地，有NPC跟你说话、关心你的状态、展开他们的故事——这种"被等待"的感觉让冰冷的功能界面变成了有温度的空间。玩家不是在和系统交互，而是"在和朋友聊天"。

#### 爽点3：好奇心驱动的期待拉力

"这次回去谁会跟我说话？""那个角色的故事接下来会发生什么？"——这种好奇心成为持续游玩的强大内驱力。不是强迫你上线，而是让你**想**上线。

#### 爽点4：渐进揭示的成就满足

棋子故事线不是一次解锁完的——像追剧一样，每次对局都可能推进一点剧情。从"我要攒碎片解锁这个棋子"变成"我要了解这个棋子的全部故事"。

#### 爽点5：投入情感的深度绑定

当你为一个棋子投入了情感、了解了他的故事，即使新版本来了更强的新棋子，你也会更有意愿继续使用你的"老朋友"。这种情感绑定形成了超越数值的忠诚。

**情绪类型**：叙事救赎 + 情感牵绊 + 渐进揭示 + 好奇心拉力

## 抽象玩法块公式

> **原本胜负循环是二进制终点（赢→奖励→再开始 / 输→惩罚→再开始），现在胜负是双轨进度的分叉点（赢→战斗力推进 + 叙事推进 / 输→叙事推进为主），返回基地不再是暂停而是核心循环的叙事消费节点。**

拆解为可迁移组件：
1. **"失败即进度"**：原本失败=负反馈终止，现在失败=推进平行进度轨
2. **"枢纽即内容"**：原本基地=功能菜单，现在基地=叙事消费空间
3. **"角色即奖励"**：原本奖励=掉落/数值，现在奖励=角色关系+故事解锁
4. **"双轨解耦"**：原本只有单一进度维度，现在战斗力与叙事两条轨道独立且互补

## MCGG迁移方案

### 羁绊效果

#### 羁绊1：叙事羁绊「同舟共济」
- **触发条件**：棋盘上有至少3个来自同一地区的棋子（如德玛西亚、艾欧尼亚等）
- **效果**：
  - 每场战斗结束后，无论胜负，获得1层"战地故事"点数
  - 累积3/6/9层时，为该地区全体棋子解锁一项永久属性加成（+5%攻击力/+10%生命值/+15%技能伤害）
  - 同时推进对应地区的叙事段落（棋手大厅专属对话）
- **设计意图**：鼓励玩家使用同地区阵容，同时将"打多局"转化为有意义的叙事积累

#### 羁绊2：关系羁绊「老友记」
- **触发条件**：棋手与每个棋子有独立的"羁绊值"，可通过以下方式积累：
  - 每局使用该棋子参与≥5场战斗 → +1羁绊值
  - 该棋子作为主C打出最高伤害 → 额外+1羁绊值
- **效果**（羁绊值阈值）：
  - Lv.3：解锁棋子专属语音台词
  - Lv.5：解锁棋子备用外观/皮肤
  - Lv.10：棋子获得专属技能强化效果（如额外伤害、特殊效果）
- **设计意图**：将棋子从"数值单位"升维为"有故事的伙伴"，形成情感绑定

### 海克斯/Augment

#### 银/金/彩：归途的期待
- **银**：当回合失败时，额外获得棋手经验+2，且下一回合商店刷新免费1次
- **金**：当回合失败时，额外获得棋手经验+3，且下一回合商店刷新免费2次
- **彩**：当回合失败时，额外获得棋手经验+4，且下一回合商店刷新免费3次 + 额外金币+2
- **设计意图**：叙事化败局补偿——输了不代表白打，下一局起点更高

#### 银/金：老友来访
- **效果**：备战阶段有X%概率触发随机一位已拥有棋子的"来访事件"
- **银**：触发概率15%，获得金币+2
- **金**：触发概率25%，获得金币+4 或 免费刷新一次
- **设计意图**：将Hades的"角色遭遇"嵌入备战阶段，制造惊喜时刻

#### 彩：命运双轨
- **效果**：
  - 胜利时：获得常规金币奖励 + 1层"战地故事"
  - 失败时：获得额外重Roll机会+1 + 2层"战地故事" + 轻微血量保护（额外少扣1血）
- **设计意图**：让胜负都有收益，消除败局的纯粹挫败感

### 棋子技能

#### 5费棋子「叙事编织者」
- **技能名**：「战友羁绊」
- **效果**：释放技能时，如果本局中该棋子已参与超过5场战斗，技能伤害额外提升30%；如果已参与超过10场战斗，技能附带"鼓舞"效果——为全体友军恢复5%最大生命值
- **设计意图**：用技能效果具象化"羁绊值"概念——用得越久越强，体现情感积累

#### 改造棋子建议：将现有某棋子技能加入"羁绊深度"维度
- **改造点**：选择已有"陪伴型"设定的棋子，为其技能增加"本局战斗中，该棋子在场回合数越多，技能效果越强"的机制
- **示例**：如果某坦克棋子在场8回合以上，其护盾技能额外吸收20%伤害
- **设计意图**：无需新增棋子，低成本嵌入Hades-like理念

## 说明

- **直接可迁移**（适配度>0.8）：海克斯系统最容易落地，纯数值/概率调整
- **需要改造**（适配度0.6-0.8）：羁绊系统和棋子技能需要设计配套的叙事内容
- **不适合**（适配度<0.6）：完整的Hub叙事空间（成本太高，不适合竞技向自走棋的核心定位）
- **核心迁移策略**：优先从海克斯入手，轻量验证"失败即进度"的叙事化思路，再逐步扩展到羁绊和棋子层
