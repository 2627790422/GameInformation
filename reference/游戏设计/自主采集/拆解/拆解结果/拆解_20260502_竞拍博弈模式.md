# 拆解结果：竞拍博弈模式

**来源**：王者万象棋（腾讯）
**机制名称**：竞拍博弈模式（Auction Bidding Mode）
**拆解时间**：2026-05-02
**执行者**：芙芙
**MCGG适配度**：0.62

---

## 基本信息

- **研究来源**：`../研究/研究结果/研究_20260430_2026自走棋新机制研究.md#机制5`
- **机制类型**：社交博弈型（Social Bidding）
- **核心变化**：从"各自刷各自的牌"→"抢同一张牌的直接博弈"

---

## 四层拆解

### 行为层

**核心变化**：从"平行单人游戏"→"直接资源争夺"

| 维度 | 传统自走棋 | 竞拍博弈 |
|------|-----------|---------|
| 玩家关系 | 间接竞争（抢公共牌库） | 直接竞争（抢同一张牌） |
| 经济用途 | D牌+升人口 | D牌+升人口+竞拍 |
| 互动频率 | 低（战斗结算看结果） | 高（竞拍轮实时互动） |
| 信息博弈 | 看对手阵容猜测 | 看对手出价博弈 |

**关键行为转变**：竞拍将自走棋从"平行竞技"推向"面对面博弈"，增加了直接的心理对抗维度。

---

### 系统层

#### 系统1：竞拍流程

| 阶段 | 内容 | 时长 |
|------|------|------|
| 展示阶段 | 公开竞拍棋子 | 5秒 |
| 出价阶段 | 存活玩家同时出价 | 10秒 |
| 揭晓阶段 | 公布胜者+扣除金币 | 3秒 |
| 结算阶段 | 胜者获得棋子，败者损失出价 | — |

#### 系统2：竞拍规则设计

| 规则 | 设计考量 |
|------|---------|
| 暗拍（密封出价） | 无法看到对手出价，增加心理博弈 |
| 明拍（递增出价） | 实时看到对手出价，增加紧张感 |
| 混合模式 | 暗拍→进入明拍加时 |
| 出价上限 | 防止all-in破坏游戏体验 |
| 失败惩罚 | 出价是否返还？建议部分返还（50%） |

**推荐方案**：暗拍（密封出价）适合移动端快节奏，避免明拍的时间消耗。

#### 系统3：竞拍触发时机

| 时机 | 棋子类型 | 频率 |
|------|---------|------|
| 固定轮次 | 4-1, 5-1, 6-1 | 每局3次 |
| 特殊事件 | 传说棋子/限定棋子 | 随机 |
| 全员投票 | 玩家投票决定是否竞拍 | 民主机制 |

---

### 决策层

#### 竞拍决策矩阵

| 决策 | 考量因素 |
|------|---------|
| 是否参与 | 这个棋子值不值得花钱？ |
| 出价多少 | 对手会出多少？我的心理价位？ |
| 策略性弃权 | 故意不出价，消耗对手金币 |
| 虚张声势 | 上轮高价中标后，本轮故意不出价 |

#### 经济系统影响

| 维度 | 影响 |
|------|------|
| 金币用途 | 新增"竞拍预算"子类别 |
| 经济节奏 | 竞拍轮次前后需要预留金币 |
| 贫富差距 | 经济领先者竞拍优势更大 |
| 翻盘机会 | 逆风方通过精准竞拍获取关键棋子 |

#### 博弈深度

| 层次 | 内容 |
|------|------|
| 一级博弈 | "我需要这个棋子" |
| 二级博弈 | "对手需要这个棋子，所以我出价消耗他" |
| 三级博弈 | "对手知道我消耗他，所以我反而出低价" |
| 四级博弈 | 循环嵌套…… |

### 情感层

#### 爽感分析

| 爽点 | 为什么爽 |
|------|---------|
| **拍赢的爽** | 在多人争夺中胜出，获得"我比你聪明"的优越感 |
| **心理博弈的爽** | 虚张声势成功/识破对手意图时的智力满足 |
| **资源掠夺的爽** | 从对手手中抢走他急需的棋子 |
| **逆袭的爽** | 劣势方通过精准竞拍拿到关键棋子翻盘 |

#### 情绪类型

- **紧张 (Tension)**：竞拍计时中的心跳加速
- **优越 (Superiority)**：胜过对手的智力快感
- **博弈 (Gambling)**：猜中对手出价的赌博感
- **怨恨 (Resentment)**（负面）：被抢走关键棋子的愤怒

#### 潜在风险（重要！）

| 风险 | 严重度 | 说明 |
|------|--------|------|
| **挫败感** | ⭐⭐⭐⭐⭐ | 竞拍失败+金币损失=双重打击，极易导致退游 |
| **金钱游戏质疑** | ⭐⭐⭐⭐ | 经济领先者竞拍优势大，可能加剧"富者愈富" |
| **社交压力** | ⭐⭐⭐ | 不想参与竞拍的玩家可能感到被强迫 |
| **节奏打断** | ⭐⭐⭐ | 竞拍轮中断自走棋流畅节奏 |
| **学习门槛** | ⭐⭐⭐ | 博弈论概念对新玩家不友好 |

## MCGG迁移方案

### 作为娱乐模式（非竞技模式）

```text
定位：仅限娱乐模式/特殊活动
原因：
  1. 适配度最低（0.62）
  2. 挫败风险高
  3. 不适合竞技排位
```

### 娱乐模式：拍卖行大乱斗

| 参数 | 设计 |
|------|------|
| 模式名称 | 拍卖行大乱斗 |
| 竞拍频率 | 每3回合一次 |
| 棋子池 | 所有棋子+限定传说棋子 |
| 初始金币 | 每玩家额外+20金币（竞拍专用） |
| 失败惩罚 | 出价返还50% |
| 特殊规则 | 竞拍传说棋子（仅此模式出现） |

### 保护机制

| 机制 | 目的 |
|------|------|
| 新手保护 | 首次参与竞拍的玩家获得"新手出价加成" |
| 连败保护 | 竞拍连续失败3次→下次竞拍获得折扣 |
| 匿名竞拍 | 不显示对手ID，减少社交压力 |
| 跳过选项 | 允许玩家选择"不参与本轮竞拍" |

### 棋子设计

**拍卖限定棋子：黑市商人（4费）**

- **仅在竞拍模式出现**
- **技能：暗箱操作**——出价时可查看对手出价区间
- **设计意图**：为竞拍模式添加独有的策略深度

## 设计总结

| 维度 | 核心洞察 |
|------|---------|
| **行为层** | 竞拍将自走棋从"平行竞技"升级为"面对面博弈"，但可能破坏原有节奏 |
| **系统层** | 暗拍（密封出价）最适合移动端，明拍增加紧张感但延长对局时间 |
| **决策层** | 多层博弈嵌套提供极高策略上限，但学习门槛也相应提高 |
| **情感层** | 最大风险是"竞拍失败+金币损失"的双重挫败，需要保护机制 |

**最终建议**：仅作为娱乐模式尝试。核心竞技模式不建议引入竞拍，会破坏自走棋的流畅感和公平性。
