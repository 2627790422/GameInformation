# 拆解结果：任务解锁机制（百子时代）

**来源**：云顶之弈S16
**机制名称**：任务解锁机制 / 百子时代（100 Champions Era）
**拆解时间**：2026-05-02
**执行者**：芙芙
**MCGG适配度**：0.75

---

## 基本信息

- **研究来源**：`../研究/研究结果/研究_20260430_2026自走棋新机制研究.md#机制3`
- **机制类型**：条件解锁型（Conditional Unlock）
- **核心变化**：从"卡池全开放，靠运气D牌"→"完成任务解锁专属棋子"

---

## 四层拆解

### 行为层

**核心变化**：从"D牌运气决定阵容"→"任务选择决定阵容"

| 维度 | 传统D牌 | 任务解锁 |
|------|---------|---------|
| 阵容决定 | D到什么牌，玩什么阵容 | 做了什么任务，解锁什么棋子 |
| 前期行为 | 攒钱→D牌 | 攒钱+完成任务 |
| 信息处理 | 看牌库，看对手 | 看任务，看解锁进度 |
| 失败处理 | "这把没D到，下把" | "这把任务没完成，但解锁进度保留了" |

**关键行为转变**：任务解锁把自走棋的"单局独立"模式，引入了"跨局进度"的概念——即使这局输了，任务进度也可能推进了，让失败不再完全是负体验。

---

### 系统层

#### 系统1：任务类型矩阵

| 任务类型 | 条件示例 | 奖励类型 | 设计意图 |
|---------|---------|---------|---------|
| 连胜任务 | 连胜3回合 | 解锁强力棋子 | 奖励优势方 |
| 连败任务 | 连败3回合 | 解锁翻盘棋子 | 给逆风方出路 |
| 经济任务 | 存到50金币 | 解锁经济棋子 | 奖励运营 |
| 战斗任务 | 单棋子击杀≥3 | 解锁战斗棋子 | 奖励操作 |
| 羁绊任务 | 激活4羁绊 | 解锁羁绊核心 | 奖励阵容构建 |

**设计意图**：任务覆盖多种玩法风格，每种风格都有专属解锁路径，避免"所有人都做同一种任务"。

#### 系统2：解锁与D牌的关系

| 原本 | 改为 |
|------|------|
| 所有棋子都在公共池 | 部分棋子需任务解锁后才进入个人牌库 |
| D牌=纯运气 | D牌=任务解锁后的"确定性收益" |
| 同行抢同一张牌 | 不同任务解锁不同牌，减少正面冲突 |

#### 系统3：任务进度系统

| 维度 | 设计 |
|------|------|
| 进度保留 | 对局结束后任务进度保留（跨局） |
| 进度重置 | 某些任务每局重置（增加紧迫感） |
| 任务链 | 完成任务A→解锁任务B→最终奖励 |
| 进度上限 | 每人最多同时进行3个任务 |

---

### 决策层

#### 短期决策（每局）

- **任务选择**：选哪个任务？连胜（激进）还是连败（稳妥）？
- **资源分配**：多少资源用于完成任务 vs 常规运营？
- **任务冲突处理**：两个任务目标矛盾时如何取舍？
- **解锁时机**：是否等任务完成再D牌？

#### 长期决策（跨局）

- **任务专精**：专精某类任务（如"连胜流任务专家"）
- **赛季任务规划**：如何安排赛季任务完成顺序
- **反制策略**：如何针对做特定任务的对手

#### 决策变化

| 传统决策 | 任务解锁后新增决策 |
|---------|------------------|
| "这回合D还是存钱" | + "这回合做任务还是存钱" |
| "D什么棋子" | + "做哪个任务来解锁什么棋子" |
| "看对手阵容" | + "看对手任务进度猜测阵容" |

---

### 情感层

#### 爽点分析

| 爽点 | 为什么爽 |
|------|---------|
| **确定性收获** | 完成任务的奖励是确定的，不像D牌纯运气 |
| **进度感** | "任务进度条在涨"的持续正反馈 |
| **解锁惊喜** | 解锁新棋子时"下一个是什么？"的期待 |
| **专属感** | "这个棋子只有解锁了任务的玩家才有"的稀缺荣耀 |
| **失败有收获** | 即使输了，任务进度也在推进，减少挫败感 |

#### 情绪类型

- **目标感 (Purpose)**：每局有明确的任务目标
- **成就感 (Achievement)**：任务完成瞬间的满足
- **进步感 (Progress)**：跨局进度积累的成长感
- **期待 (Anticipation)**：解锁新棋子前的悬念

#### 潜在风险

- **任务疲劳**：重复做同样的任务会腻
- **Pay-to-Win风险**：是否可能通过付费跳过/加速任务
- **"上班感"**：如果任务变成每日打卡，失去游戏乐趣
- **平衡风险**：任务奖励棋子过强会破坏公平性

## MCGG迁移方案

### 任务系统设计

#### 核心任务表（5条主线）

| 任务线 | 条件 | 阶段奖励 |
|--------|------|---------|
| **征服者之路** | 累计胜场 | 5胜→稀有棋子 / 10胜→史诗棋子 / 20胜→传说棋子 |
| **收藏家之旅** | 收集不同棋子 | 10种→金币 / 20种→稀有装备 / 30种→传说装备 |
| **羁绊大师** | 激活所有羁绊 | 5种→海克斯 / 10种→专属棋子 / 全羁绊→传说皮肤 |
| **经济大亨** | 累计存钱 | 30金→利息+1% / 50金→金币加成 / 100金→特殊棋子 |
| **冒险家日志** | 使用不同城邦/模式 | 3种→经验 / 5种→海克斯 / 全模式→限定外观 |

### 棋子设计

**任务限定棋子：时空旅者（5费）**

- **解锁条件**：完成任意两条任务主线
- **技能：时空裂缝**——对最远敌人造成300/500/800伤害，若击杀，重置技能CD
- **设计意图**：高门槛解锁 → 高回报棋子，让完成任务的价值肉眼可见

### 海克斯

- **任务加速（Gold）**：本局任务进度加倍
- **双线作战（Prismatic）**：可以同时进行4个任务（原上限3个）
- **速通达人（Silver）**：完成任务条件降低20%

## 设计总结

| 维度 | 核心洞察 |
|------|---------|
| **行为层** | 任务解锁将"D牌运气"转化为"任务选择"，让玩家有更多主动决策空间 |
| **系统层** | 跨局进度系统打破"单局独立"的局限，让失败也有收获 |
| **决策层** | 每局新增"做任务 vs 常规运营"的战略权衡 |
| **情感层** | 最大价值在于"失败的正反馈"——输了也在前进，大幅减少流失 |
