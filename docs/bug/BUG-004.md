# BUG-004: 游戏模块收藏星标不显示

**状态**: fixed
**日期**: 2026-06-28
**严重程度**: major
**来源**: 用户报告

## 现象
登录后切入 AI 模块可看到文章卡片右上角的收藏星标，切换到游戏模块后星标不显示。

## 根因
`card()` 函数渲染星标时默认 `display:none`。`refreshBookmarkStars()` 负责根据登录状态显示星标，但**只在 `onAuthChange` 回调中触发**（登录/登出时）。模块切换 → `loadArts()` 渲染新卡片 → 没有调用 `refreshBookmarkStars()` → 星标保持隐藏。详情页的 `loadDetail()` 同理。

## 修复
在 `loadArts()` 和 `loadDetail()` 的渲染完成后追加 `refreshBookmarkStars()`，确保每次重新渲染卡片时星标显示状态与当前 auth 同步。

### 修改文件
- `web-viewer/public/app.js` — `loadArts()` 末尾 + `loadDetail()` 末尾各加一行 `refreshBookmarkStars()`
