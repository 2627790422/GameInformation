四方案对比

A — 朱砂戳记
左上角 · 衬线体 · 空心红框 · -6°旋转 · 报纸印章美学
👍 最独特，完全不像 badge。👎 绝对定位可能跟紧凑卡片冲突，旋转在暗色下可读性略降
B — 实心橙红
card-top 行尾 · 等宽字体 · 纯色填充 #e74c1b · 光圈脉冲 · 闪烁圆点
👍 最醒目，视觉权重高。👎 可能过于抢眼（但可调饱和度）
C — 左侧色条
4px 橙红竖线替代灰边 · 行内文字 NEW · 整卡变色
👍 面积最大最明显，结构清晰。👎 跟 pipeline 彩色顶线和灰色左边框设计打架（卡片本身就是 border-left:none + 彩色 top）
D — 暖色底纹
渐变暖橙背景 · "✦ 新到" 衬线体 · 无边框
👍 优雅低调，最符合温暖纸媒。👎 不够醒目（你说的问题没解决），且渐变在大量卡片上可能不统一

🎯 建议

方案 A（朱砂戳记） 最符合"顶尖资讯"的报纸编辑美学，独一无二不跟任何 badge 撞脸。
如果觉得印章太激进，方案 B（实心橙红） 是折中——保持药丸形状但用高饱和实心填充 + 脉冲光圈拉开差距。
